# Правила проекта

## Удаление
- Ничего нельзя удалять без явного разрешения пользователя. Перед любым удалением (файлов, папок, данных, строк кода) — сначала объяснить на естественном языке, что именно и зачем планируется удалить, и дождаться подтверждения.

## Написание кода
- Перед написанием кода — сначала описать план на естественном языке, чтобы пользователь понял, что будет сделано, и подтвердил.
- Если код может что-то удалить (файлы, данные, записи и т.д.) — обязательно спросить у пользователя на естественном языке, выполнять это или нет. Не выполнять без подтверждения.

## Документация проекта (Этап 0)

- `docs/mathcad2000_reference.md` — справочник по Mathcad 2000: формат MCD, синтаксис, встроенные функции
- `docs/algorithm_description.md` — детальное описание алгоритма обработки прописей: все этапы, формулы, параметры, коэффициенты растворимости, формулы SolutionMaker
- `docs/ershov_dissertation_summary.md` — конспект диссертации Ершова: теория интерферометрии, мёртвая зона, кинетика Аррениуса, влияние примесей
- `docs/research_improvements.md` — обзор литературы и предложения по улучшению: современные методы, BCF-модель, автоматизация, обновление эталонов
