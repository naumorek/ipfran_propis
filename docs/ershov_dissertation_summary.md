## Глава 1: Теория роста кристаллов KDP

### Основные положения

- KDP — тетрагональная сингония, пространственная группа I4̄2d
- Рост происходит по механизму слоевого роста (BCF — Burton-Cabrera-Frank)
- Скорость роста различна для граней {101} и {100}

### Исторический контекст

- Первые работы по KDP: 1962 год
- Типичные скорости: KDP ~1 мм/день, DKDP ~1.5 мм/день
- Основные центры исследований: LLNL (Lawrence Livermore National Laboratory)

### Ключевые зависимости

- Скорость роста пропорциональна перенасыщению в степени m (m ≈ 2 для KDP)
- Существует критическое перенасыщение σ_c (мёртвая зона) ниже которого рост не наблюдается
- Мёртвая зона зависит от концентрации примесей, особенно Fe³⁺

### Оптические измерения

- Длины волн светодиодов: **λ = 470 нм** и **λ = 590 нм**

- Критический угол: θ > 0.53°

### Методика

- Интерферометрический метод позволяет измерять in-situ скорость роста кристалла
- Два канала (два светодиода) обеспечивают избыточность и контроль
- Разрешение по скорости: определяется периодом осцилляций (~λ/2n за полупериод)

### Характеристики DKDP

- DKDP Phase II при λ = 590 нм
- Отличия от KDP: другая растворимость, другие кинетические параметры
- D₂O чистота: 99.5–99.7%

## Глава 3: Интерферометрический метод (ключевая глава)

### Основные формулы

**(3.1) Скорость роста:**
```
p(r) = R / (V_st(r) · ρ)
```
где R — скорость роста, V_st — скорость ступени, ρ — плотность

**(3.2) Перенасыщение:**
```
σ_s = (C_s - C_0) / C_0
```
где C_s — концентрация раствора, C_0 — равновесная концентрация

**(3.6) Кинетическое уравнение роста:**
Связывает скорость роста с перенасыщением через кинетические коэффициенты

**(3.13) Интерферометрическое определение скорости:**
```
U(λ)^(-1/2) ∝ (σ_sat - σ(ζ)) / R
```

**(3.14) Число Грасгофа (конвекция):**
```
Gr = β·D·L³·C·ρ·g·x^(3/5) / (ν²·J³)
```
Определяет режим массопереноса (диффузионный vs конвективный)

### Концепция мёртвой зоны

- Мёртвая зона возникает когда локальное перенасыщение σ(r) < σ_c
- Характеризуется числом Дамкёлера p = (скорость кинетики) / (скорость диффузии)
- При p < 1: диффузионно-ограниченный рост
- При p > 1: кинетически-ограниченный рост
- Мёртвая зона увеличивается с ростом концентрации примесей (особенно Fe³⁺)

## Глава 4: Экспериментальные результаты

### Кинетика роста (формула Аррениуса)

**(4.1):**
```
R = A · exp(-B / (T - T₀))
```

Подгоночные параметры для KDP, грань {101}, 9.8% кислоты:
- B = 1.662 × 10¹² (?)
- T₀ = 7184.3 K (?)
- Диапазон температур: 36.5–38°C

### Сравнение с данными SolutionMaker

В SolutionMaker используется формула:
```
R = 9.11×10¹³ · exp(-8336.42 / (T + 273.15)) · σ²
```

Это согласуется с формулой (4.1) из диссертации:
- A = 9.11×10¹³
- B = 8336.42 K
- Показатель при σ: n = 2

- Чистота D₂O: 99.5–99.7%

## Ключевые идеи для модернизации алгоритма

1. **Интерферометрический метод** хорошо описан и обоснован теоретически — фундамент алгоритма корректен

2. **Мёртвая зона** — ключевой параметр качества раствора, определяемый примесями Fe³⁺. Связь количественная: больше Fe → больше мёртвая зона → хуже раствор

3. **Число Дамкёлера** позволяет определить лимитирующий фактор (диффузия или кинетика) — это можно использовать для более точного фиттинга

4. **Формула Аррениуса** с σ² — основная модель кинетики, согласуется между диссертацией и SolutionMaker

5. **Два светодиода** (470 нм и 590 нм) дают две независимые оценки скорости роста — в модернизированном алгоритме можно использовать это для оценки погрешности

6. **Эталонные кривые** (Cfe=0 и Cfe=16 ppm) основаны на данных диссертации и привязаны к конкретным условиям — необходимо обновить с учётом новых данных
