## ОСТАВШИЕСЯ МЕЛКИЕ ОТЛИЧИЯ

### 1. x0 в zs-формуле — разная опорная температура

**Mathcad**: `x0 = co₀ - co₁·T[m-1]` (температура в КОНЦЕ sub-array)
**Python**: `x0 = co₀ - co₁·T[im]` (температура в начале мёртвой зоны)

x0 входит только в малый коррекционный член L1·x0·(1/x2-1/x1). При ΔT≈5°C → Δ(co₁·T)≈10⁻⁴ → влияние на скорость ~0.2%.

**Файл**: classic.py:458-460

### 2. Растворительные экстремумы — другой метод

**Mathcad**: тот же zero-crossing с y0s = mean(S1[isat1..m-1]) в зоне растворения
**Python**: scipy.find_peaks + detrend

Не критично — растворительный фит (s0_d, s1_d) вторичный параметр.

### 3. Линейная регрессия zs1 (Ls1) для растворения

**Mathcad**: строит регрессию R(t) в зоне dmin..dmax для растворения
**Python**: не реализовано (не влияет на основные выходные параметры)
