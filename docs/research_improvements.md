# Исследование: современные методы и предложения по улучшению

## 1. Современные методы измерения скорости роста кристаллов

### Интерферометрические методы

Классический метод, используемый на стенде ИПФ РАН, основан на **двухлучевой интерферометрии** с двумя светодиодами (λ = 470 нм и 590 нм). Это надёжный подход, но существуют более современные альтернативы:

**Ортогональная поляризационная интерферометрия (OPI)** — позволяет измерять неоднородности структуры кристалла, связанные с отклонением от фазового синхронизма. Используется для оценки оптического качества быстро выращенных KDP.

**Интерферометр Маха-Цандера** — применяется для онлайн-измерения концентрации и её градиента в процессе роста.

**Интерферометр Майкельсона** — используется для визуализации спирального механизма роста на пирамидальных гранях.

**Высокоразрешающая интерферометрия** — разрешение изменений высоты в 200 Å на гранях шириной несколько сотен мкм. Основана на обработке фазовой информации из интерференционного поля интенсивности с коррекциями от опорного сигнала (Land et al., 1994).

### Цифровая обработка интерферограмм

**Метод Такеды (Fourier Transform Method, 1982)** — автоматическая обработка интерферограмм без подвижных компонентов:
1. БПФ интерференционной картины
2. Выделение одной боковой полосы в спектре
3. Обратное БПФ → получение обёрнутой фазы
4. Развёртка фазы (phase unwrapping)
- **Преимущество перед текущим алгоритмом:** не требует ручной детекции экстремумов, даёт непрерывную фазу

**Вейвлет-анализ:** в отличие от Фурье-анализа, даёт локальные значения амплитуды и фазы для каждой гармоники. Позволяет изучать нестационарные сигналы — что особенно важно для переменной скорости роста.

**Сравнение с текущим алгоритмом:**
| Параметр | Текущий (Mathcad) | Фурье-метод | Вейвлет-метод |
|----------|-------------------|-------------|---------------|
| Детекция скорости | По экстремумам | По непрерывной фазе | По мгновенной частоте |
| Разрешение | 1 полоса | Субполосное | Субполосное |
| Шум | Медианная фильтрация | Частотная фильтрация | Многомасштабная |
| Автоматизация | Ручная | Полная | Полная |
| Нестационарность | Плохо | Средне (STFT) | Хорошо |

## 2. Исследования KDP/DKDP в ведущих лабораториях

### LLNL (Lawrence Livermore National Laboratory, США)

- **Наталия Зайцева** — привезла технологию быстрого роста из России в LLNL в 1993 году
- Технология быстрого роста из точечной затравки получила R&D 100 Award (1994)
- Кристаллы, которые традиционным методом растили 2 года, выращиваются за **2 месяца**
- Кристаллы весом **до 360 кг** (800 фунтов)
- NIF требует ~480 оптических элементов из KDP и DKDP
- Кристаллизатор: 2000 л, Holden-тип, стабильность ±0.02°C
- Скорости роста: **10–25 мм/день** в направлениях [001] и [100]

### Методология LLNL

- Затравка: точечная (~1 см³) или длинная (20×20×260 мм)
- Программа температуры: 60→35°C
- Перенасыщение: **3–10%** в зависимости от температуры
- Вращение: 10–15 об/мин
- Длительность: 70–100 дней
- Результат: кристаллы 471×480×400 мм, 254 кг
- Оптическое качество быстро выращенных кристаллов **не уступает** традиционным

### Китай (SG-III)

- Выращен кристалл KDP апертурой **600 мм** методом быстрого роста
- KDP до 47 см из кристаллизатора 1000 л со скоростью 10–25 мм/день

### DKDP для мощных лазеров

- Большие DKDP кристаллы с дейтерированием 70% для оптики >400 мм
- Два метода: традиционный (1–2 мм/день, >2 лет) и быстрый (10–40 мм/день, 2–3 месяца)
- Ключевая проблема: контроль дейтерирования при высоких скоростях

## 3. Растворимость KDP и DKDP

### Общая формула

```
C(T) = k₀ + k₁·T + k₂·T²
```

где C — массовая доля соли, T — температура в °C.

### Данные из литературы (г KH₂PO₄ на 100 г H₂O)

Классическая формула (Mullin, 2001):
```
C*(T) ≈ 10.68 + 0.3616·T + 0.00506·T²     (г/100 г H₂O)
```

**Примечание:** В SolutionMaker используется массовая доля (безразмерная), а не г/100 г H₂O.

### Сравнение с коэффициентами SolutionMaker

Коэффициенты из SolutionMaker (массовая доля):
- KDP нейтр: C = 0.123 + 2.719×10⁻³·T + 1.1087×10⁻⁵·T²
- При T = 50°C: C = 0.123 + 0.136 + 0.028 = 0.287 (28.7%)
- Литературные данные при 50°C: ~33 г/100 г → C ≈ 0.248

**Вывод:** Необходима верификация коэффициентов. Возможные причины расхождения:
1. Разные единицы (г/100 г H₂O vs массовая доля в растворе)
2. Влияние кислоты на растворимость
3. Старые данные из советских справочников

### Монография Рашковича

**Л.Н. Рашкович**, *KDP Family Single Crystals* (Adam-Hilger, 1991; CRC Press, 2021) — основной справочник по:
- Механизмам и кинетике кристаллизации
- Морфологии поверхности
- Дислокационным спиралям
- Влиянию примесей
- Методам создания перенасыщения

## 4. Мёртвая зона (метастабильная зона)

### Механизм Кабрера-Вермилье (Cabrera-Vermilyea)

Примесные частицы адсорбируются на террасах между ступенями и действуют как «стопоры»:
- Если расстояние между стопорами < 2r₀ (удвоенный критический радиус), ступень останавливается
- Дальнейшее движение возможно только при увеличении перенасыщения σ > σ_d
- **Мёртвая зона:** σ_d = 2Ωα·Cᵢ^(1/2) / kT

где Ω — молекулярный объём, α — удельная поверхностная энергия, Cᵢ — концентрация примеси.

### Выход из мёртвой зоны

De Yoreo et al. (Nature, 1998) показали:
- При малых σ рост полностью подавлен
- При увеличении σ рост возобновляется через движение **макроступеней** (пучков элементарных ступеней)
- Отдельные ступени и малые пучки остаются неподвижными
- Традиционные модели не объясняют это поведение — нужно учитывать макроступени

### Двойное действие примесей (Alexandru et al., 1995)

- При **высоком перенасыщении**: примеси стимулируют двумерное зародышеобразование
- При **малом перенасыщении**: блокируют ступени дислокационного механизма
- Граница мёртвой зоны зависит от концентрации примеси **и pH раствора**

### Метастабильная зона в быстром росте

Зайцева и др. (LLNL): метастабильная зона до **40°C** в ширину, индукционный период >3 месяцев в перемешиваемых растворах. Позволяет скорости 10–40 мм/день без спонтанной нуклеации.

### EDTA как хелатирующий агент

Добавление 1 вес.% EDTA:
- Связывает ионы тяжёлых металлов (Fe³⁺, Cr³⁺, Al³⁺) в крупные комплексы
- Увеличивает метастабильную зону
- Не встраивается в структуру кристалла (молекула слишком большая)

## 5. Кинетические модели роста KDP

### Модель BCF (Burton-Cabrera-Frank, 1951)

Рост граней через винтовые дислокации:
- При малых σ: R ∝ σ² (параболический закон)
- При больших σ: R ∝ σ (линейный закон)
- Переход при σ ~ 19σ₁ (где σ₁ зависит от температуры)

**Применение к KDP (Nicolau, 1974):**
- Анализ данных роста граней {101} при 50°C
- Определены характеристические константы BCF-функции
- Рассчитаны приближённые энергии активации для дегидратации, десорбции, поверхностной диффузии

### Модель в текущем алгоритме

```
G(σ) = s₀ · (σ - s₁)^w      (w обычно = 1)
```

Это **упрощённая степенная модель с мёртвой зоной**, не полная BCF.

### Формула Аррениуса из SolutionMaker

```
R = 9.11×10¹³ · exp(-8336.42 / (T + 273.15)) · σ²
```

Это согласуется с параболическим режимом BCF (R ∝ σ²) при малых перенасыщениях с температурной зависимостью кинетического коэффициента по Аррениусу.

### Современные уточнения

- **2025:** Исследование влияния изменений перенасыщения на рост граней {100} KDP — механизм роста не зависит от предыстории для исследованного диапазона σ
- **2022:** Модель «stoppers» для примесей — учёт макроступеней и временнóй шкалы адсорбции
- **Рашкович и Молдажанова (1996):** Кинетика граней бипирамиды KDP менее чувствительна к примесям, чем грани призмы

## 6. Влияние примесей Fe на рост KDP

### Концентрационные эффекты

| Концентрация Fe³⁺ | Эффект |
|-------------------|--------|
| <1 ppm | Нормальный рост (высокочистое сырьё) |
| ~5 ppm | Минимальное влияние, остаточные примеси |
| ~16 ppm | Существенное замедление (эталон «плохого» раствора в алгоритме) |
| ~20 ppm | Значительные изменения кинетики |
| ~30 ppm | Качественное изменение морфологии — «суперступени» высотой до 1500 нм |
| >60 ppm | Кристаллы можно использовать как УФ-фильтры |
| ~100 ppm | Сильная кинетическая блокировка |

### Механизмы

1. **Адсорбция на кинках** (узлах излома) граней {100} и {101}
2. Образование **коллоидных агрегатов** трёхвалентных металлов в растворе
3. **Стопорный эффект** (Cabrera-Vermilyea): расстояние между адсорбированными частицами определяет σ_d
4. Эффект **ослабевает с температурой**: при высоких T мёртвая зона уменьшается

### Морфология при разных [Fe³⁺]

- **0 ppm:** Гладкие ступени, спиральный рост
- **5–16 ppm:** Макроступени, умеренная шероховатость
- **30 ppm:** «Суперступени» из сотен элементарных ступеней, «пальцевая» нестабильность

## 7. Быстрый рост крупных кристаллов

### Метод точечной затравки

Разработан в МГУ (Рашкович), усовершенствован в LLNL (Зайцева):
- Затравка ~1 см³ (точка)
- Рост на порядок быстрее традиционного (10–40 vs 1–2 мм/день)
- Не требует большой затравки
- Подходит для KDP и DKDP

### Метод длинной затравки

- Затравка 20×20×260 мм по направлению [001]
- Результат: 471×480×400 мм за 70 дней
- Преимущество: более равномерный начальный рост

### Ключевые факторы успеха

1. **Фильтрация раствора** — непрерывная для удаления микрочастиц
2. **Стабильность температуры** — ±0.02°C
3. **Программирование перенасыщения** — 3–10% в зависимости от размера кристалла
4. **Вращение** — 10–15 об/мин, реверсивное
5. **Чистота раствора** — Fe³⁺ < 1 ppm

## 8. Предложения по улучшению алгоритма

### 8.1 Автоматизация определения экстремумов

**Текущее:** Ручной подбор параметров n1, n2, im, isat.
**Предложение:** Автоматическая детекция через:
- Аналитический сигнал (преобразование Гильберта) → огибающая + мгновенная частота
- Или метод Такеды (БПФ) для непрерывной фазы
- Автоматическое определение начала/конца роста по порогу мгновенной частоты

### 8.2 Улучшенное определение мёртвой зоны

**Текущее:** Визуальный подбор «вертикального участка».
**Предложение:**
- Модель Кабрера-Вермилье для фиттинга
- Автоматическое определение σ_d по точке перегиба R(σ)
- Учёт температурной зависимости σ_d

### 8.3 Современный фиттинг кинетической кривой

**Текущее:** G(σ) = s₀·(σ - s₁)^w с ручным перебором.
**Предложение:**
- Нелинейный МНК (scipy.optimize.curve_fit или аналог)
- Модель BCF с мёртвой зоной: R(σ) = β·(σ - σ_d)² / (σ₁ + σ - σ_d) при σ > σ_d
- Статистическая оценка качества фиттинга (R², доверительные интервалы)

### 8.4 Обновление эталонных кривых

**Текущее:** Два эталона (Cfe=0 и Cfe=16 ppm), старые, полиномиальные.
**Предложение:**
- Набор эталонных кривых для нескольких концентраций Fe (0, 5, 10, 16, 20, 30 ppm)
- Параметрическая модель: σ_d(C_Fe, T) и β(C_Fe, T)
- База данных измерений для построения новых эталонов

### 8.5 Обработка двух каналов

**Текущее:** Каналы обрабатываются параллельно, но не объединяются.
**Предложение:**
- Оценка погрешности через расхождение каналов
- Взвешенное усреднение с учётом качества сигнала в каждом канале
- Автоматическое исключение паразитных полос (замена «прочитай.txt»)

### 8.6 Верификация коэффициентов растворимости

**Текущее:** Коэффициенты из SolutionMaker могут быть устаревшими.
**Предложение:**
- Сравнение с данными Рашковича (1991), Зайцевой (1995), и современными публикациями
- Возможность пользователю задавать свои коэффициенты
- Валидация по экспериментальным точкам

### 8.7 Пакетная обработка

**Текущее:** Один MCD-файл = один PRN-файл, полностью ручной процесс.
**Предложение:**
- Пакетная обработка всех PRN файлов в директории
- Автоматический отчёт с таблицей параметров (Mat) для каждого файла
- Сравнительные графики между пропись-файлами

### 8.8 Решение известной проблемы расхождения tn

**Проблема:** Расхождение между точкой насыщения на стенде и в кристаллизаторе.
**Предложения:**
- Калибровочная кривая tn_стенд → tn_кристаллизатор по историческим данным
- Учёт разницы в гидродинамике (перемешивание, конвекция)
- Возможность ввода корректировки оператором
