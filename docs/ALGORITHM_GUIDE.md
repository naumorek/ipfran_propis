## 1. Что мы измеряем

Установка: кювета с раствором KDP/DKDP, два LED-канала (λ₁ ≈ 470 нм, λ₂ ≈ 590 нм), призменный делитель, детектор.

**Входные данные** — PRN файл (~65000 строк, 1 отсчёт/сек):
- Столбец 1: LED1 (В) — интерференционный сигнал канала 1
- Столбец 2: LED2 (В) — интерференционный сигнал канала 2
- Столбец 4: Температура (raw, ~273 + T°C)
- Столбец 6: Время (HH:MM:SS)

**Один цикл** ≈ 2-2.5 часа: охлаждение (рост) → мёртвая зона → нагрев (растворение).

**Ключевые параметры из MCD файла** (задаются вручную для каждого цикла):
- `n1, n2` — границы цикла в PRN файле
- `im1, isat1` — грубые границы мёртвой зоны и растворения
- `im, isat` — точные границы

## 2. Конвейер обработки (пошагово)

### Шаг 1: Сигнал
```
signal = (LED1 - vt1) / (LED2 - vt2)
```
Отношение каналов нормализует дрейф источника света. `vt1 ≈ vt2 ≈ 0.0001` — фоновые пороги.

### Шаг 2: Базовая линия
```
S1 = medsmooth(signal, 5)     — медианный фильтр
y0 = mean(S1[0..im1])         — среднее в зоне роста
```

### Шаг 3: Детекция экстремумов (функция `e`)
Каждый экстремум = λ/4 ≈ 0.12 мкм толщины кристалла.

**Типичный файл: 10-30 экстремумов** = 10-30 реальных точек роста.

Алгоритм:
1. Zero-crossing detection (шаг 2): находим где сигнал пересекает y0
2. В каждом полупериоде: максимум отклонения |signal - y0|

### Шаг 4: Фаза (arcsin интерполяция)
Из 17 экстремумов → 2297 точек фазы на d-step сетке (d = 3.3 отсчёта):
```
phase[i] = π/2·(n-1) + arcsin((signal[i] - e[n-1]) / (e[n] - e[n-1]))
```

**Это ключевая интерполяция**: 17 реальных → ~2300 точек.

### Шаг 5: Кумулятивная толщина L
```
x = co₀ - co₁·T          — оптический коэффициент
L[i] = (phase[i]/(π·x) + L0·x0/x - L0) · nn
```
L = толщина кристалла в полосах. L0 ≈ 1000-4000 (начальная толщина).

### Шаг 6: LOESS сглаживание L → L_smooth
```
L_smooth = LOESS(L, span=0.15, degree=2)
```
Local quadratic regression (алгоритм Cleveland). Убирает arcsin-осцилляции.

### Шаг 7: Скорость роста R
```
R[i] = (L_smooth[i+1] - L_smooth[i]) / dt · 30 · nn     [мкм/мин]
R_mm_day = R × 1.441                                       [мм/день]
```

**Итого: из 17 реальных точек → 2296 точек скорости через arcsin + LOESS.**

### Шаг 8: Пересыщение σ%
```
Cn = C(tn),  Cm = C(T)          — растворимость при tn и текущей T
σ% = 100 · ln(Cn / Cm)          — относительное пересыщение
```

### Шаг 9: LOESS кривая F1(σ)
```
F1 = LOESS(R, σ%, span=0.2, degree=2)
```
Второе LOESS-сглаживание — финальная кинетическая кривая.

### Шаг 10: Выходные параметры

| Параметр | Формула | Единицы | Что означает |
|----------|---------|---------|-------------|
| te | a[isat, 4] - 273.15 | °C | Температура начала растворения |
| tn | te - 0.01·(te-25) | °C | Температура насыщения |
| Td | te - T[im] | °C | Ширина мёртвой зоны |
| Sigm | 100·ln(Cn/Cm) | % | Мёртвая зона в пересыщении |
| s1 | Grid search | % | Порог мёртвой зоны (power law) |
| s2 | -Q1/Q2 | % | Форм-фактор (√R vs σ) |
| Sig035 | F1(σ) > 0.35 мкм/мин | % | σ при R = 0.5 мм/день |

## 4. Что реально измеряется, а что дорисовывается

### Реальные данные (из эксперимента):
- **~8000 отсчётов** сигнала LED1/LED2 (1/сек)
- **17 экстремумов** (интерференционных полос) = **17 реальных точек** роста кристалла
- Температура T(t) — непрерывная
- te, tn — из температуры

### Интерполяция и сглаживание (алгоритм дорисовывает):
1. **17 → 2297**: arcsin-интерполяция фазы между экстремумами
2. **2297 → 2297**: фаза → L (пересчёт через оптические коэффициенты)
3. **2297 → 2297**: LOESS сглаживание L → L_smooth (span=0.15)
4. **2297 → 2296**: dL_smooth/dt → скорость R (производная)
5. **2296 → кривая**: LOESS сглаживание R(σ) → F1 (span=0.2)

**Итого**: из 17 реальных точек алгоритм генерирует 2296 точек R через двойное LOESS сглаживание.

### Эталонные кривые (Si, Si1):
- **8 точек** каждая — полиномиальная аппроксимация R(dT) для стандартных условий
- 4 набора коэффициентов: ±кислота × ±железо
- Вычисляются при te текущего эксперимента

## 7. Формулы (для справки)

**Растворимость**: `C(T) = co₂ + co₃·T + co₄·T²`

**Пересыщение**: `σ = 100 · ln(C(tn) / C(T))`

**Переохлаждение**: `dT = tn - T`

**Скорость (power law)**: `G(σ) = s0 · (σ - s1)^w`

**Скорость (Аррениус)**: `R = 9.11×10¹³ · exp(-8336.42/(T+273.15)) · σ²`

**s2**: `√R = Q1 + Q2·σ` → `s2 = -Q1/Q2` (σ при R=0)

**Sig035**: первый σ где LOESS(R) > 0.35 мкм/мин = 0.504 мм/день
