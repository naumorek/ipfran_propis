# Анализ точности DIY инструментации кристаллизатора

## Цель

Строгая оценка: достаточна ли точность DIY датчиков (330–560 EUR) для получения осмысленных R(σ) из кристаллизатора? Полное распространение ошибок по цепочке κ → C → σ → R(σ).

---

## 1. Исходные физические данные

### 1.1 Растворимость KDP (CRC Handbook)

| T (°C) | C_sat (г/100г H₂O) |
|---------|-------------------|
| 40      | 25.28             |
| 50      | 28.95             |
| 60      | 32.76             |

**При T = 50°C:**
```
C_sat = 28.95 г/100г H₂O
dC_sat/dT = (32.76 − 25.28) / 20 = 0.374 г/(100г·°C)
```

### 1.2 Проводимость KDP раствора

При рабочих концентрациях (~25–35 wt%): κ ≈ 25–80 мСм/см при 25°C.

Чувствительности (оценка из данных CRC + Belruss 1984):
```
dκ/dC ≈ 1–3 мСм/см на wt%   (зависит от концентрации)
dκ/dT ≈ +1.5–2.5% / °C      (стандартно для электролитов)
```

### 1.3 Скорости роста KDP при разных σ

Данные: ACS Omega (2025, PMC 11800150), грань {100}, T ≈ 25°C:

| σ (%) | R (нм/с) | R (мм/день) |
|-------|----------|-------------|
| 6.2   | 2.5      | 0.22        |
| 8.3   | 8.7      | 0.75        |
| 10.4  | 17.3     | 1.49        |
| 12.6  | 24.9     | 2.15        |
| 14.7  | 34.5     | 2.98        |

Параболический закон при малых σ: R ≈ A·σ², A ≈ 0.065 нм/с на %².

**Экстраполяция на малые σ:**

| σ (%) | R (мм/день) |
|-------|-------------|
| 0.5   | ~0.001      |
| 1.0   | ~0.006      |
| 2.0   | ~0.022      |
| 3.0   | ~0.050      |
| 5.0   | ~0.14       |

**В кристаллизаторе (быстрый рост, Zaitseva LLNL):** σ = 3–10%, R = 5–15 мм/день. Наш диапазон: R = 2.5–4 мм/день → σ ≈ 5–8%.

## 2. Распространение ошибок: κ, T → C → σ

### 2.1 Пересыщение

Определение:
```
σ = (C − C_sat(T)) / C_sat(T) × 100%
```

Дифференцируя:
```
δσ = √[ (∂σ/∂C · δC)² + (∂σ/∂T · δT)² ]

∂σ/∂C = 100% / C_sat
∂σ/∂T = −100% · (dC_sat/dT) / C_sat    (при фиксированном C: нагрев → снижение σ)
```

При T = 50°C, C_sat = 28.95:
```
∂σ/∂C = 100 / 28.95 = 3.45 %/(г/100г)
∂σ/∂T = −100 × 0.374 / 28.95 = −1.29 %/°C
```

### 2.2 Вклады ошибок

| Источник | δ параметра | Вклад в δσ | Формула |
|----------|------------|-----------|---------|
| **Температура** (MAX31865) | δT = 0.03°C | **0.039%** | 1.29 × 0.03 |
| **Температура** (класс A Pt100) | δT = 0.25°C | **0.32%** | 1.29 × 0.25 |
| **Концентрация** (DIY κ, лучший случай) | δC = 0.1 г/100г | **0.35%** | 3.45 × 0.1 |
| **Концентрация** (DIY κ, типичный) | δC = 0.3 г/100г | **1.03%** | 3.45 × 0.3 |
| **Концентрация** (Belruss LLNL) | δC = 0.05 г/100г | **0.17%** | 3.45 × 0.05 |

### 2.3 Суммарная ошибка δσ

| Сценарий | δT | δC | **δσ** | Можно ли разрешить σ_dead = 0.3%? |
|----------|-----|-----|--------|-----------------------------------|
| **DIY оптимистичный** | 0.03°C | 0.1 г/100г | **0.35%** | НЕТ (δσ > σ_dead) |
| **DIY типичный** | 0.03°C | 0.3 г/100г | **1.03%** | НЕТ |
| **Промышленный (Belruss)** | 0.03°C | 0.05 г/100г | **0.18%** | На пределе |
| **Промышленный + калибровка** | 0.01°C | 0.03 г/100г | **0.11%** | ДА |

### 2.4 Вывод по пересыщению

**DIY проводимость НЕ разрешает мёртвую зону (σ_dead ≈ 0.3%).** Ошибка δσ ≈ 0.35–1.0% сравнима или больше σ_dead.

**НО:** в кристаллизаторе мы работаем при σ = 3–10% (R = 2.5–4 мм/день). При σ = 5%:
```
Относительная ошибка = δσ/σ = 0.35/5 = 7%    (DIY оптимист.)
                              = 1.03/5 = 21%   (DIY типичный)
```

**7% относительная ошибка пересыщения — приемлемо** для мониторинга процесса и построения грубой R(σ). Мёртвую зону не увидим, но рабочий диапазон — да.

## 3. Ошибка проводимости: откуда берётся δC

### 3.1 Источники ошибки в κ → C

```
C = f(κ, T)   — калибровочная поверхность

δC = √[ (∂C/∂κ · δκ)² + (∂C/∂T_κ · δT_κ)² + δC_cal² ]
```

| Источник | Величина | Вклад в δC |
|----------|---------|-----------|
| **Шум АЦП** (ADS1115, PGA×8, 128 SPS, 100 отсчётов) | δκ/κ ≈ 0.01% | ~0.003 г/100г — пренебрежимо |
| **Дрейф электродов** (нерж. сталь, без платинирования) | δκ/κ ≈ 1–3% за неделю | **0.3–1.0 г/100г** — ГЛАВНЫЙ источник |
| **Дрейф электродов** (платинированная Pt) | δκ/κ ≈ 0.1–0.3% за неделю | 0.03–0.1 г/100г |
| **Температура при κ** (dκ/dT ≈ 2%/°C, δT = 0.03°C) | δκ/κ ≈ 0.06% | ~0.02 г/100г — мало |
| **Калибровка** (polyfit R² = 0.999, 54 точки) | δC_cal ≈ 0.05–0.1 г/100г | 0.05–0.1 г/100г |
| **Загрязнение раствора** (примеси меняют κ) | неконтролируемо | 0.1–0.5 г/100г |

### 3.2 Критический вывод

**Главная проблема — не электроника, а электроды.** АЦП даёт 0.01%, а дрейф нержавеющих электродов — 1–3% за неделю. При росте кристалла 30–60 дней это катастрофа.

**Решения:**
1. **Платинированные электроды** — снижают дрейф до 0.1–0.3%/неделю → δC ≈ 0.1 г/100г
2. **Еженедельная рекалибровка** — измерить κ эталонного раствора → скорректировать offset
3. **Двухточечная коррекция** — два эталонных раствора (C₁ и C₂) → slope + offset каждую неделю

### 3.3 Влияние примесей на проводимость

Раствор KDP не чистый. Fe³⁺, Cr³⁺, Al³⁺ и другие ионы вносят свой вклад в κ. При C_Fe = 10 ppm = 10⁻³ г/100г: вклад в κ пренебрежим (< 0.01%). **Примеси не проблема** для кондуктометрии.

Но **pH** влияет на проводимость: H⁺ имеет аномально высокую подвижность. Если pH дрейфует на 0.5 единицы за рост, это может внести ошибку δκ/κ ≈ 1–2%. Решение: одновременный мониторинг pH (датчик #2) + коррекция.

## 4. Ошибка камеры: R(t) из размеров кристалла

### 4.1 Разрешение камеры

5 МП камера (2592 × 1944 пикселей):

| Объектив | Расстояние | Поле зрения | Размер пикселя |
|----------|-----------|------------|---------------|
| 16 мм | 500 мм | 178 × 134 мм | 0.069 мм/пкс |
| 16 мм | 1000 мм | 356 × 268 мм | 0.137 мм/пкс |
| 50 мм | 1000 мм | 114 × 86 мм | 0.044 мм/пкс |

Субпиксельная точность детекции края (OpenCV Canny + Hough): **0.1–0.3 пикселя**.
Лучшие методы (Zernike moments): **0.03 пикселя**.

### 4.2 Точность измерения размера кристалла

С объективом 16 мм на расстоянии 1 м, субпиксель 0.1:
```
δL_edge = 0.137 × 0.1 = 0.014 мм на одну грань
δL_кристалл = √2 × 0.014 = 0.020 мм   (две грани)
```

### 4.3 Точность скорости роста

```
R = ΔL / Δt

δR = δL / Δt = √2 × δL_edge / Δt
```

| R (мм/день) | R за 1 час | SNR (δL = 0.020 мм) | SNR (δL = 0.050 мм) |
|-------------|-----------|---------------------|---------------------|
| 1.0         | 0.042     | 2.1 — **маргинально** | 0.8 — **нет** |
| 2.5         | 0.104     | 5.2 — **OK** | 2.1 — **маргинально** |
| 4.0         | 0.167     | 8.3 — **хорошо** | 3.3 — **OK** |
| 10.0        | 0.417     | 20.8 — **отлично** | 8.3 — **хорошо** |

При R = 2.5–4 мм/день (наш диапазон) и δL = 0.020 мм: **SNR = 5–8 за 1 час**. Приемлемо.

### 4.4 Усреднение за 3 часа (как обходы)

За 3 часа при R = 3 мм/день: ΔL = 0.375 мм, δL = 0.020 мм → **SNR = 19**. Отлично.

### 4.5 Особенности кристаллизатора высотой 2.5–3 м

- Камера сбоку через окно: расстояние до кристалла зависит от положения
- **Перспективное искажение:** если кристалл не в центре поля зрения, масштаб неоднороден → ошибка до 1–3%
- **Решение:** телецентрический объектив (дороже) или калибровка по мишени на расстоянии кристалла
- **Качание питателя (7 сек):** кристалл может колебаться → смаз. Решение: выдержка < 1/50 с (LED импульс)

### 4.6 Вывод по камере

**Камера даёт δR ≈ 0.3–0.5 мм/день** при часовом интервале и **δR ≈ 0.1 мм/день** при 3-часовом. В рабочем диапазоне R = 2.5–4 мм/день — **относительная ошибка 3–15%**. Приемлемо.

## 5. Комбинированная ошибка R(σ)

### 5.1 Что мы получаем

Из камеры: R(t) ± δR
Из проводимости + T: σ(t) ± δσ

Точки (σᵢ, Rᵢ) на кинетической кривой R(σ).

### 5.2 Сценарии

**Сценарий A: DIY оптимистичный (платинированные электроды + еженедельная калибровка)**
```
δσ = 0.35% (абсолютная)
δR = 0.3 мм/день (часовой) или 0.1 мм/день (3-часовой)
```

**Сценарий B: DIY типичный (нержавейка без частой калибровки)**
```
δσ = 1.0%
δR = 0.5 мм/день
```

**Сценарий C: Промышленный (Belruss-метод)**
```
δσ = 0.18%
δR = 0.1 мм/день
```

### 5.3 Что можно и нельзя увидеть

| Задача | Нужна δσ | Нужна δR | DIY-A | DIY-B | Промышл. |
|--------|---------|---------|-------|-------|----------|
| Мёртвая зона (σ_dead ≈ 0.3%) | < 0.1% | — | **НЕТ** | **НЕТ** | На пределе |
| R(σ) в зоне роста (σ = 3–10%) | < 1% | < 0.5 мм/д | **ДА** | **ДА** | ДА |
| Сравнение β_крист vs β_стенд | < 1% | < 0.5 мм/д | **ДА** | Грубо | ДА |
| Мониторинг деградации раствора | < 2% | < 1 мм/д | **ДА** | **ДА** | ДА |
| Оптимизация программы охлаждения | < 1% | < 0.5 мм/д | **ДА** | Грубо | ДА |
| Обнаружение спонтанной нуклеации | < 3% | — | **ДА** | **ДА** | ДА |

### 5.4 Графическая иллюстрация

Типичная R(σ) для нашего раствора (из стенда):
```
σ:    0    0.3   1.0   2.0   3.0   5.0   7.0   10.0
R:    0    0     0.01  0.05  0.15  0.6   1.5   3.5   (мм/день)
```

С DIY-A (δσ = 0.35%, δR = 0.3 мм/д) — что реально видим:
```
σ = 5.0 ± 0.35%  →  R = 0.6 ± 0.3 мм/д   (SNR = 2, видим рост)
σ = 7.0 ± 0.35%  →  R = 1.5 ± 0.3 мм/д   (SNR = 5, хорошо)
σ = 10.0 ± 0.35% →  R = 3.5 ± 0.3 мм/д   (SNR = 12, отлично)
σ = 3.0 ± 0.35%  →  R = 0.15 ± 0.3 мм/д  (SNR = 0.5, НЕ ВИДИМ)
σ = 1.0 ± 0.35%  →  R = 0.01 ± 0.3 мм/д  (SNR ≈ 0, не видим)
```

**Реальная рабочая зона DIY: σ > 5%, R > 0.5 мм/день.**

## 6. Влияние конструкции кристаллизатора

### 6.1 Градиент температуры по высоте (2.5–3 м)

Типичный градиент в кристаллизаторе с естественной конвекцией: 0.5–2°C/м.
При высоте 3 м: **ΔT = 1.5–6°C** между верхом и низом.

Если датчик T и датчик κ стоят в разных точках:
```
ΔT_датчики = 0.5°C   (допустим, разнесены на 0.5 м)
→ ошибка в σ: 1.29 × 0.5 = 0.65%   ← КАТАСТРОФА
```

**Решение:** датчик T и ячейка κ должны быть **рядом** (< 10 см). И оба — рядом с кристаллом.

### 6.2 Качание питателя (период 7 с)

V-образный питатель с лопастями создаёт периодическое перемешивание. Раствор у кристалла осциллирует по концентрации и температуре:
```
δT_качание ≈ 0.1–0.5°C (оценка для активного перемешивания)
δC_качание ≈ 0.05–0.2 г/100г
```

**Решение:** усреднение за N периодов. При периоде 7 с и записи 1 Гц:
```
За 70 сек (10 периодов): δT_средн ≈ δT_качание / √10 ≈ 0.03–0.15°C
За 7 мин (60 периодов): δT_средн ≈ δT_качание / √60 ≈ 0.01–0.06°C
```

**Усреднение за 5–10 минут решает проблему качания.**

### 6.3 Обеднённый слой у кристалла

Раствор у поверхности кристалла обеднён (кристалл потребляет KDP). Локальное пересыщение **у кристалла** ниже, чем в объёме:
```
σ_поверхн = σ_объём − R / k_d
```

где k_d = D/δ — коэффициент массопереноса, D ≈ 5×10⁻⁶ см²/с (KDP), δ ≈ 50–500 мкм (зависит от конвекции).

При R = 3 мм/день = 3.5×10⁻⁶ см/с, k_d = 5×10⁻⁶/0.01 = 5×10⁻⁴ см/с:
```
Δσ_BL ≈ R / k_d = 3.5×10⁻⁶ / 5×10⁻⁴ = 0.007 → 0.7%
```

**Пограничный слой даёт систематическое занижение σ на ~0.7%.** Это физический эффект, не ошибка измерения. Но он означает, что R(σ_объём) из кристаллизатора будет лежать **ниже** R(σ_стенд), потому что в объёме σ выше чем у кристалла.

## 7. Честная оценка: что получим и что нет

### 7.1 Что DIY инструментация ДАСТ (при аккуратной работе)

1. **Тренд σ(t)** — видим как пересыщение меняется в процессе роста (десятки часов). Абсолютная точность ±0.5%, но **тренд** виден с точностью ±0.1% (дрейф электродов медленный).

2. **Грубая R(σ) при σ > 5%** — 5–10 точек на кривой за один рост (30 дней). Достаточно для оценки β_крист и сравнения с β_стенд.

3. **Мониторинг аномалий** — спонтанная нуклеация (резкое падение σ), перегрев (σ < 0), остановка роста.

4. **pH(t) при рабочей T** — дрейф за месяц, корреляция с качеством кристалла.

5. **R(t) по граням** — с точностью ±0.3 мм/день (часовой) или ±0.1 мм/день (3-часовой).

### 7.2 Что DIY инструментация НЕ ДАСТ

1. **Мёртвую зону из кристаллизатора** — δσ > σ_dead. Для σ_dead по-прежнему нужен стенд прописей.

2. **Точную R(σ) при σ < 3%** — SNR < 1 и для камеры, и для проводимости.

3. **Локальное σ у поверхности кристалла** — проводимость измеряет объём, не пограничный слой.

4. **Абсолютное значение σ лучше ±0.5%** — без промышленных датчиков.

### 7.3 Что нужно для бо́льшей точности

| Улучшение | Стоимость | Эффект на δσ |
|-----------|-----------|-------------|
| Платинированные электроды | +50 EUR (материал) | 1.0% → 0.35% |
| Промышленная κ-ячейка (E+H) | +3000 EUR | 0.35% → 0.17% |
| Рекалибровка 2 раза/неделю | 0 EUR (время) | Убирает дрейф |
| Pt100 1/10 DIN + индивид. калибровка | +100 EUR | δT: 0.03 → 0.01°C |
| Датчики κ и T рядом (< 10 см) | 0 EUR (монтаж) | Убирает градиент |
| Усреднение 5–10 мин | 0 EUR (софт) | Убирает качание |

## 8. Рекомендация

### Стоит ли делать DIY?

**Да, но с реалистичными ожиданиями:**

- **Для мониторинга процесса** (σ > 3%, R > 0.5 мм/день): DIY достаточно. Это уже огромный шаг вперёд по сравнению с "вслепую".
- **Для кинетической кривой R(σ)**: получим 5–10 точек при σ = 3–10%. Грубая, но сравнимая со стендом.
- **Для мёртвой зоны**: нет. Стенд прописей незаменим.
- **Для оптимизации охлаждения**: да. Видеть σ в реальном времени → держать R в оптимуме.

### Приоритет доработок (бесплатно, сразу)

1. **Датчики κ и T монтировать рядом** (< 10 см) и рядом с кристаллом
2. **Усреднение 5–10 мин** в софте (гасит качание питателя)
3. **Еженедельная рекалибровка** κ по эталонному раствору
4. **Платинирование электродов** (если есть H₂PtCl₆)

### Что даст максимальный эффект за минимальные деньги

**Одна промышленная κ-ячейка** (3000 EUR) + DIY всё остальное → δσ ≈ 0.2% → можно видеть переходную зону σ = 1–3%. Это единственная покупка, которая качественно меняет картину.
