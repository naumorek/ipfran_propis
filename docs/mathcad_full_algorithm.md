## 2. Детекция экстремумов (скр. 1, 22-23)

### 2a. Грубая детекция — нахождение zero-crossings для растворения (extrs)

```
y0 := 1/(im1) · Σ(i=0..im1-1) S1_i        — среднее S1 в зоне роста
y0s := 1/(m-isat1) · Σ(i=isat1..m-1) S1_i  — среднее S1 в зоне растворения

extrs :=
  ma ← -10
  mb ← -10
  k ← 1   if S1_{isat} - y0s > 0
  k ← -1  otherwise
  j ← 0
  for i ∈ isat1, isat1+2 .. m-1        ← ШАГ 2!
    (ma ← i)   if √(S1_i - y0s)² > mb
    mb ← √(S1_i - y0s)²  if √(S1_i - y0s)² > mb
    (mx_{j,0} ← ma)   if (S1_i - y0s)·k < 0
    (mx_{j,1} ← S1_{ma})  if (S1_i - y0s)·k < 0
    j ← (j+1)  if (S1_i - y0s)·k < 0
    (mb ← -1)  if (S1_i - y0s)·k < 0
    k ← -k     if (S1_i - y0s)·k < 0
    (mx_{j,0} ← ma)  if m-1-ma > 4
    (mx_{j,1} ← mb)  if m-1-ma > 4
  mx
```

**Логика:** Идём от isat1 до конца с шагом 2. Отслеживаем максимальное отклонение от y0s.
При смене знака (S1_i - y0s)·k < 0 — записываем позицию и значение максимума в полуволне.
Это находит позиции ЭКСТРЕМУМОВ (максимумов и минимумов).

### 2b. Фильтрация крайних экстремумов:
```
kk := rows(extrs) - 1
— Удалить первый, если слишком близко к isat1:

— Удалить последний, если слишком близко к концу:

— Удалить последний, если значение ≤ 0:
```

### 4) Разбиение синусоиды на части и нахождение угловых координат:

```
y :=
  p ← rows(e) - 1
  n ← 0
  g ← floor(isat1/d)
  for i ∈ 0..g+1
    a_{i,0} ← i·d
    a_{i,1} ← I(a_{i,0})          — интерполированное значение сигнала
    a_{i,3} ← t(a_{i,0})          — интерполированное значение времени
    for j ∈ 0..p
      n ← j
      break if e_{j,0} > a_{i,0}
    a_{i,2} ← -asin((a_{i,1} - e_{p,1}) / (e_{1,1} - e_{p,1}))   if n < 1
    a_{i,2} ← π/2·(n-1) + asin((a_{i,1} - e_{p-1,1}) / (e_{p,1} - e_{p-1,1}))  otherwise
    s ← i
    break if a_{i,0} + d > e_{p,0}
  for i ∈ s+1..g
    break if s+1 > g
    a_{i,0} ← i·d
    a_{i,1} ← I(a_{i,0})
    a_{i,3} ← t(a_{i,0})
    a_{i,2} ← π/2·n + asin((a_{i,1} - e_{p,1}) / (e_{p-1,1} - e_{p-1,1}))
  a
```

Здесь `I(x) := linterp(tau, I, x)` — интерполяция сигнала, `t(x) := linterp(tau, t, x)` — интерполяция времени.

## 4. Точные экстремумы через квадратичную регрессию (es) (скр. 22-23)

```
es := for j ∈ 0..ks
  n ← |extrs_{j+1,0} - extrs_{j,0}| · k1   if j < ks
  n ← |extrs_{j-1,0} - extrs_{j,0}| · k1   otherwise
  n ← floor(n)
  u ← extrs_{j,0} - floor(n/2)   if j < 1
  u ← extrs_{j,0} - n            otherwise
  v ← extrs_{j,0} + n            if extrs_{j,0} + n < m-1
  v ← m - 1                      otherwise
  for i ∈ u..v
    x_{i-u,0} ← i - u
    x_{i-u,1} ← S1_i
  y ← submatrix(x, 0, v-u, 0, 1)
  z ← regress(y^(0), y^(1), 2)     — квадратичная регрессия!
  q_{j,0} ← -z_4 / (2·z_5)         — x вершины параболы
  q_{j,1} ← z_3 + z_4·q_{j,0} + z_5·(q_{j,0})²   — y вершины
  q_{j,0} ← q_{j,0} + u            — абсолютная позиция
  k ← -1  if q_{ks-1,1} < y0s
  k ← 1   otherwise
```

**Ключевое:** Для каждого грубого экстремума берём окрестность размером `n = floor(|расстояние_до_соседа| · k1)`,
где k1 = 0.15. Делаем квадратичную регрессию (полином 2-й степени) и находим точную вершину параболы.

```
ys := j ← 0

    ys_{j,0} ← floor(ys_{j,0})
```

### 5) Скорость роста и температура (z):

```
z := for i ∈ 0..rows(y)-2
  z_{i,0} ← (ys_{i,0} + ys_{i+1,0}) / 2         — средняя позиция
  y1 ← ceil(ys_{i,0})
  y2 ← floor(ys_{i+1,0})
  z_{i,2} ← 1/(y2-y1+1) · Σ_{r=y1}^{y2} (a_{r,4} - 273.15)   — средняя T
  z_{i,1} ← (T_{i+1} - T_i) / (Tau(y_{i+1,0}) - Tau(y_{i,0})) · 30·nn   — скорость!
  z
```

**Скорость R = (ΔФаза) / (ΔВремя) · 30 · nn**
- ΔФаза = T_{i+1} - T_i — разница угловых координат (в файле T — это массив из ys)
- ΔВремя = Tau(y_{i+1}) - Tau(y_i) — разница времён
- 30 = пересчёт в мм/день (или мм/мин в зависимости от единиц)
- nn = ±1 направление

### Подготовка данных:

### tel = -bs/as — линейная экстраполяция:
```

```

### Фиттинг s0, s1 степенным законом:
```
w := 1

s := min ← 100000
  for i ∈ 0..n-1
    for x ∈ VX_i, VX_i + 0.01 .. VX_{i+1}
      q1 ← Σ_{k=i}^{n} (VX1_k - x)^w · VY1_k
      q2 ← Σ_{k=i}^{n} (VX1_k - x)^{2w}
      q ← q1/q2
      f ← Σ_{k=0}^{i} (VY1_k)² + Σ_{k=i}^{n} [q·(VX1_k - x)^w - VY1_k]²
      s_0 ← q   if f < min
      s_1 ← x   if f < min
      min ← f   if f < min
  s

s = (-0.0129, -0.4305)    — s_0, s_1
```

**Ключевое:** s1 — это порог σ% мёртвой зоны (может быть отрицательным).
G_h = s_0 · (h·0.01 - s_1)^w — формула с h в процентах, но (h·0.01 - s1) означает
что s1 уже в долях? Нет: `G_h = s_0 · (h·0.01 - s_1)^w`, и `G_h = if(h·0.01 < s_1, 0, s_0·(h·0.01 - s_1)^w)`.

Подождём... h — это индекс от 0 до floor(VX_{hh}·100). Т.е. h·0.01 — это σ в процентах/100? Нет, если VX_hh ~ 6%, то floor(6·100)=600, h·0.01 = 0..6. Значит h·0.01 = σ%. И s_1 = -0.4305 означает σ в %.

### s2 — наклон в координатах √R vs σ% (скр. 7-8):
```
Q := j ← 0
  for i ∈ 0..rows(z)-1
    break if z_{i,1} < 0.0
    q_{j,0} ← z_{i,3}      if z_{i,1} < 0.3     — σ%
    q_{j,1} ← √(z_{i,1})   if z_{i,1} < 0.3     — √R
    j ← j+1                 if z_{i,1} < 0.3
  q

Q1 := intercept(Q^(0), Q^(1))     — свободный член
Q2 := slope(Q^(0), Q^(1))         — наклон

s2 := -Q1/Q2

s2 = -0.0359
```

**КРИТИЧЕСКИ ВАЖНО:**
- Q собирает точки где R > 0 И R < 0.3 (из z, НЕ из отсортированного u!)
- q_{j,0} = σ%, q_{j,1} = √R
- Линейная регрессия: √R = Q1 + Q2·σ%
- **s2 = -Q1/Q2** = -intercept/slope = точка пересечения с осью σ% (где √R = 0)
- Также: `q3_{p,0} = p`, `q3_{p,1} = p·Q2 + Q1` — модельная прямая

## Ключевые отличия от нашей реализации:

1. **s2 = -Q1/Q2** (intercept/slope в координатах √R vs σ%), НЕ наклон в log-log
2. **Sig035** — через LOESS (loess + interp), не аналитически
3. **Экстремумы** — zero-crossings с шагом 2, затем квадратичная регрессия для уточнения
4. **Угловые координаты** — непрерывная фаза через asin, не просто подсчёт экстремумов
5. **Скорость** — dФаза/dВремя с учётом показателя преломления, не просто подсчёт полос
6. **tm** имеет поправку: `a_{im,4} - 273.15 - 0.01·(te-25)`
7. **LOESS** используется для сглаживания R(σ) с span=0.2
