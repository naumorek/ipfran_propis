# Детальное описание алгоритма обработки прописей KDP/DKDP

## Общая схема

Алгоритм обрабатывает данные интерферометрического стенда для определения кинетических параметров роста кристаллов KDP/DKDP. Реализован в Mathcad 2000 Professional, каждый MCD-файл обрабатывает один PRN-файл.

**Выходные параметры:**
- Температура насыщения (`tn`, °C)
- Мёртвая зона (`Td` в °C, `s1` в %)
- Кинетическая кривая R(ΔT) — скорость роста от переохлаждения
- Сравнение с эталонными кривыми (Cfe=0 и Cfe=16 ppm)

---

## Формат входных данных (PRN)

### Структура файла

Текстовый файл, ~65000 строк, 7 столбцов, разделённых пробелами/табуляцией. Один отсчёт в секунду (~18 часов записи).

### Столбцы PRN-файла

| Столбец | Индекс в Mathcad | Физический смысл | Диапазон значений |
|---------|-------------------|-------------------|-------------------|
| 1 | `a[i,0]` | Порядковый номер отсчёта | 0 – 65000 |
| 2 | `a[i,1]` | Сигнал светодиода 1 (интерференция) | ~1–6 В |
| 3 | `a[i,2]` | Сигнал светодиода 2 (интерференция) | ~1–4 В |
| 4 | `a[i,3]` | Флаг/статус (всегда 0.00) | 0.00 |
| 5 | `a[i,4]` | Температура раствора (условные единицы, ~273+T) | 317–322 |
| 6 | `a[i,5]` | Температура раствора (°C) | 44–49 |
| 7 | `a[i,6]` | Время (HH:MM:SS) | — |

**Примечание:** Столбец 5 — вероятно, показания термометра в единицах, требующих пересчёта (317.82 ≈ 273.15 + 44.67, но не точно соответствует столбцу 6). Столбец 6 — температура в °C, используемая в алгоритме.

### Физика интерферометрического сигнала

Два светодиода с разными длинами волн (λ ≈ 470 нм и λ ≈ 590 нм) освещают грань растущего кристалла. При росте кристалла толщина слоя меняется, создавая интерференционные полосы. Каждый максимум/минимум сигнала соответствует изменению толщины на λ/(2n), где n — показатель преломления.

Скорость роста определяется по частоте осцилляций:
```
R = λ / (2n · Δt)
```
где Δt — период осцилляций сигнала.

### Первичная визуализация

Строятся графики:
1. **Сырой интерференционный сигнал** — осциллирующая кривая (синусоида с переменной амплитудой и частотой)
2. **Температура раствора** — монотонный рост от ~45°C до ~49°C

Оператор визуально определяет:
- Участок, где начинается рост (появляются осцилляции)
- Участок растворения (осцилляции прекращаются)
- Границы мёртвой зоны

## Этап 2: Определение температуры насыщения и мёртвой зоны

### Физический смысл

Эксперимент: температура раствора медленно повышается. Кристалл сначала растёт (раствор пересыщен), затем проходит точку насыщения и начинает растворяться.

- **Температура насыщения (tn)** — температура, при которой скорость роста = 0
- **Мёртвая зона (Td)** — разница температур между tn и моментом начала видимого роста

### Алгоритм

1. Выделяется участок сигнала в зоне ~6500–9000 отсчётов (зона перехода рост→растворение)
2. Применяется сглаживание (`medsmooth`, `loess` с `span1 = 0.15`)
3. Строится кривая огибающей сигнала
4. Оператор подбирает параметры `im`, `isat` так, чтобы:
   - Часть кривой стала **вертикальной** (скорость роста = 0)
   - Ноль производной определяет момент насыщения
5. Температура в точке нулевой скорости = `tn`

### Мёртвая зона

```
Td = tn - t_начала_роста        (в °C)
s1 = σ(Td)                       (в % перенасыщения)
```

Типичные значения: `Td = 0.4°C`, `s1 = -0.68%`

## Этап 3: Детекция экстремумов интерференционных полос

### Алгоритм поиска экстремумов

Программный блок в Mathcad реализует алгоритм детекции пиков и впадин синусоидального сигнала:

```
extrs := | s ← -1
          | n ← 0
          | for i ∈ 0..length-1
          |   l ← S2_i       if |S2_i - y0| > a
          |   n ← i          if |S2_i - y0| > a
          |   s ← |S2_i - y0| if |S2_i - y0| > a
          |   s ← -1
          | for i ∈ n+1..length-1
          |   l ← k+1        if S2_i = b
          |   n1 ← n1-1      if S2_i = b
```

### Обработка двух каналов

Для каждого светодиода (канал 1 и канал 2):
1. Находятся позиции максимумов и минимумов
2. Экстремумы сортируются: `u = csort(data, col)`
3. Проверяется согласованность: `Vx_j - u_j`, `Ty_j - u_j`, `span > 0.5`

### Пересчёт в скорость роста

Расстояние между соседними экстремумами (максимум → максимум или минимум → минимум) соответствует изменению толщины на λ/n:

```
ΔL = λ / (2 · n_refr)
Δt = t_экстр(i+1) - t_экстр(i)
R_i = ΔL / Δt                    (мкм/мин или мм/день)
```

Температура для каждой точки берётся как среднее между соседними экстремумами:
```
T_i = (T_экстр(i+1) + T_экстр(i)) / 2
```

## Этап 4: Фиттинг кинетической кривой

### Модель

Степенной закон роста с мёртвой зоной:

```
G(σ) = s0 · (σ·0.01 - s1)^w
```

где:
- `σ` — перенасыщение (%)
- `s0` — кинетический коэффициент
- `s1` — мёртвая зона (% перенасыщения, обычно отрицательное)
- `w` — показатель степени (обычно `w = 1`)

### Оптимизация параметров

Метод наименьших квадратов (ручная реализация перебором):

```
fit := | sum ← 100000
        | for s ∈ диапазон
        |   for x ∈ диапазон
        |     q1 ← Σ(VX_k - x)² · VY_k      // взвешенная сумма
        |     q2 ← Σ(VX_k - x)^(2w)          // нормировка
        |     q ← q1/q2
        |     f ← Σ(VY_k - G(VX_k))²         // невязка
        |     (sg ← q, s1 ← s, mm ← f)  if f < mm
```

### Параметр Sig035

Определяет перенасыщение, при котором нормированная функция F1(x) > 0.35:

```
Sig035 := | for i ∈ 0..rows(u)-1
           |   DZ ← VX_i
           |   break if F(VX_i) > 0.35
           | DZ
```

Типичное значение: `Sig035 = 2.2` (%)

### Линейная регрессия для проверки

```
Q1 = intercept(Q)     // Q1 = -0.3533 × 10⁻³
Q2 = slope(Q)          // Q2 = -0.6833
```

## Этап 5: Эталонные кривые и итоговые результаты

### Эталонные кинетические кривые

Полиномиальные кривые для сравнения, зависящие от температуры:

```
Si(j, te) = a(j,0) + a(j,1)·te + a(j,2)·te²
```

где `j = 0..7` — индекс параметра, `te` — равновесная температура.

Две эталонные кривые:
- **Cfe = 0** (зелёная) — чистый раствор, идеальная кинетика
- **Cfe = 16 ppm** (красная) — загрязнённый раствор, подавленная кинетика

### Интерпретация графика R(ΔT)

- Ось X: переохлаждение ΔT = tn - T (°C)
- Ось Y: скорость роста R (мм/день)
- **Левее зелёной** → отличный раствор
- **Между зелёной и красной** → нормальный раствор
- **Правее красной** → плохой раствор, требует очистки

### Альтернативное представление R(σ)

- Ось X: перенасыщение σ (%)
- Ось Y: скорость роста R (мкм/мин)
- Показывает мёртвую зону (s1) и параметр формы (s2)

### Итоговая матрица параметров (Mat)

```
Mat = (Salt, Acid, te, te1, tn, s1, Sigm, s2, Sig035, Td, n1, n2, ww, d, dtau, l, isat1, im, isat, im1)
```

Пример значений:
| Параметр | Значение | Описание |
|----------|----------|----------|
| Salt | 1 | KDP |
| Acid | 0 | Нейтральный раствор |
| te | 48.85 | Равновесная температура (°C) |
| tn | 48.6 | Температура насыщения (°C) |
| Td | 0.4 | Мёртвая зона (°C) |
| s1 | -0.68 | Мёртвая зона (% перенасыщения) |
| Sigm | 0.5 | Параметр σ |
| s2 | 0.27 | Параметр формы кинетической кривой |
| Sig035 | 2.2 | Порог σ при F > 0.35 |

## Формулы SolutionMaker (раздел 0.4)

### Приготовление нового раствора

Дано: объём V (мл), температура насыщения Tn (°C), кислотность h (%).

```
C = k0 + k1·Tn + k2·Tn²                                    // растворимость

m_вода = V · ρ · (1-C) · (1 - h/Cac) / (1 - (e/d)·C)     // масса воды (если кислый)
m_вода = V · ρ · (1-C) / (1 - (e/d)·C)                     // масса воды (если нейтральный)
m_кислота = m_вода / (Cac/h - 1)                            // масса кислоты
m_соль = (m_вода + m_кислота) / (1/C - 1)                   // масса соли
```

### Изменение температуры насыщения

Добавление соли при повышении Tn:
```
C0 = k0 + k1·Tn0 + k2·Tn0²
C1 = k0 + k1·Tn1 + k2·Tn1²
M0 = V · ρ / (1 - (e/d)·C0)                    // полная масса раствора
Δm_соль = (C1 - C0) · M0 / (1 - C1)            // масса добавляемой соли
```

### Определение перенасыщения

```
σ = ln(C / C_eq)                                // перенасыщение (%)
```

где C — фактическая концентрация, C_eq — равновесная при данной температуре.

### Кинетика роста (формула Аррениуса)

```
R = A · exp(-B / (T + 273.15)) · σ²
```

где:
- `A = 9.11×10¹³` (предэкспоненциальный множитель)
- `B = 8336.42` (энергия активации / R_газ, в K)
- `σ` — перенасыщение
- `R` — скорость роста

**Логарифмическая форма:**
```
ln(R) = ln(A) - B/(T + 273.15) + 2·ln(σ)
ln(A) ≈ 30.229
```

### Расчёт скорости опускания температуры

Для поддержания постоянной скорости роста R при росте кристалла:

```
Для каждого шага по высоте h:
  C(h) = (C0·M0 - α·(h + h1 - h0)) / (M0 - α·(h + h1 - h0))
  Te = (C(h) - k0) / k1                        // равновесная температура
  σ₁ = exp(0.5·(ln(R) - ln(A) + B/(273.15 + Te)))
  C₁ = C / exp(σ₁)
  t₁ = (C₁ - k0) / k1
  p = (g - t) / (Δh / R)                       // скорость опускания (°C/час)
  τ = 3600 / (216 · p)                          // время шага (сек)
```

где `α = ρ_кр · s1 · s2 · 0.001` — фактор площади сечения кристалла.

## Пошаговый процесс обработки (сводка)

```
1. Загрузка PRN → матрица a[m×7]
2. Выбор рабочего участка [n1..n2]
3. Извлечение сигналов LED1 = a[*,1], LED2 = a[*,2], T = a[*,5]
4. Сглаживание сигналов (medsmooth, loess)
5. Визуальный подбор границ im, isat (оператор)
6. Определение tn по нулю производной сглаженной огибающей
7. Определение Td = tn - T_начала_роста
8. Детекция экстремумов (максимумы/минимумы) обоих каналов
9. Пересчёт экстремумов в R(T) — скорость роста от температуры
10. Пересчёт T → ΔT = tn - T (переохлаждение)
11. Пересчёт T → σ (перенасыщение через формулу растворимости)
12. Фиттинг G(σ) = s0·(σ - s1)^w методом наименьших квадратов
13. Определение параметров s1, s2, Sig035
14. Построение кинетической кривой R(ΔT)
15. Наложение эталонных кривых (Cfe=0, Cfe=16 ppm)
16. Формирование итоговой матрицы Mat
```
