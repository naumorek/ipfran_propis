# Часть I: Физика и распространение ошибок

## 1. Распространение ошибки температуры насыщения

### 1.1 Цепочка пересыщение → скорость

Растворимость KDP описывается полиномом:

```
C₀(T) = k₀ + k₁·T + k₂·T²
```

Для KDP neutral (Mullin & Amatavivadhana; ACS Omega 2025):
- k₀ = 0.17554, k₁ = 0.00102, k₂ = 0.0000743 (кг KDP / кг H₂O, T в °C)

Альтернативные коэффициенты (из Mathcad ИПФ РАН):
- k₀ = 0.123, k₁ = 2.719×10⁻³, k₂ = 1.1087×10⁻⁵

Логарифмическое пересыщение:

```
σ = 100 · ln(C(tₙ) / C(T))
```

### 1.2 Аналитическая производная ∂σ/∂tₙ

По правилу дифференцирования:

```
∂σ/∂tₙ = 100 · C'(tₙ) / C(tₙ)
```

где C'(T) = k₁ + 2·k₂·T.

**Числовой пример** при tₙ = 49°C (коэффициенты ИПФ РАН):
- C(49) = 0.123 + 2.719×10⁻³·49 + 1.1087×10⁻⁵·49² = 0.2833
- C'(49) = 2.719×10⁻³ + 2·1.1087×10⁻⁵·49 = 3.806×10⁻³

```
∂σ/∂tₙ = 100 · 3.806×10⁻³ / 0.2833 = 1.34 %/°C
```

**Числовой пример** (коэффициенты Mullin, T=30°C):
- C(30) = 0.17554 + 0.00102·30 + 0.0000743·900 = 0.273
- C'(30) = 0.00102 + 2·0.0000743·30 = 0.005478
- ∂σ/∂tₙ = 100 · 0.005478 / 0.273 = 2.01 %/°C

### 1.3 Пропагация в скорость роста R

Для степенного закона R = s₀·(σ - s₁)^w:

```
∂R/∂σ = s₀ · w · (σ - s₁)^(w-1)
```

Комбинируя через chain rule:

```
∂R/∂tₙ = s₀ · w · (σ - s₁)^(w-1) · 100 · C'(tₙ) / C(tₙ)
```

Относительная ошибка скорости:

```
δR/R = w · δσ / (σ - s₁)
```

**Числовой пример**: w=2, σ=5%, s₁=1%, δσ=0.2% (от δtₙ=0.15°C):

```
δR/R = 2 · 0.2 / (5 - 1) = 10%
```

**Вывод**: Даже умеренная ошибка δtₙ = 0.15°C даёт ~10% ошибку скорости роста. При δtₙ = 0.3°C (типичная ручная ошибка) — 20%.

### 1.4 Пропагация в параметры мёртвой зоны

```
Td = te - T[im]
Sigm = 100 · ln(C(te) / C(T[im]))
```

Чувствительность Sigm к ошибке в im:

```
∂Sigm/∂T_im = -100 · C'(T_im) / C(T_im) ≈ -1.34 %/°C
```

При скорости температурного хода ~0.001°C/отсчёт и ошибке δim = ±50 отсчётов:
- δT_im = ±0.05°C
- δSigm = 1.34 · 0.05 = ±0.07% (15-20% от типичного Sigm = 0.3-0.5%)

### 1.5 Полный бюджет неопределённости

| Источник | Типичная ошибка | Вклад в δσ | Вклад в δR/R |
|----------|----------------|------------|--------------|
| δtₙ (ручной) | ±0.3°C | ±0.40% | ±20% |
| δtₙ (auto) | ±0.1°C | ±0.13% | ±7% |
| δT_sensor | ±0.01°C | ±0.013% | ±0.7% |
| δC (полином) | ±1% | ±0.01% | ±0.5% |
| δim (ручной) | ±50 отсч. | — | Td ±15% |
| **Доминирует** | **δtₙ** | | |

- MIT: Methods for estimating supersaturation: https://dspace.mit.edu/bitstream/handle/1721.1/124866/c9ce00843h.pdf

## 2. Модель Кабреры-Вермильи (CV) — полный вывод

### 2.1 Физическая постановка

Прямая ступень продвигается по поверхности кристалла, на которой адсорбированы неподвижные примесные частицы со средним расстоянием L.

### 2.2 Эффект Гиббса-Томсона для искривлённой ступени

Равновесие искривлённой ступени радиуса ρ:

```
Δμ(ρ) = Δμ_∞ · (1 - r_c/ρ)
```

Критический радиус для 2D нуклеации:

```
r_c = γ·Ω / (k_B·T·σ)
```

где:
- γ — свободная энергия ступени на единицу длины (Дж/м)
- Ω — молекулярный объём (~9.7×10⁻²⁹ м³ для KDP)
- k_B·T — тепловая энергия
- σ — относительное пересыщение

### 2.3 Скорость ступени между пиннинг-центрами

Сегмент ступени, зажатый между двумя примесями на расстоянии L, может продвигаться только если кривизна не превышает критическую:

```
v_step = v₀ · (1 - 2·r_c / L)
```

где v₀ — скорость ступени на чистой поверхности.

**Когда L < 2·r_c**, ступень полностью заблокирована: v_step = 0.

Более точно: для ступени, проходящей через «ворота» шириной L, максимальная кривизна при выпучивании даёт радиус ρ = L/2. Ступень пройдёт только если L/2 > r_c, т.е. L > 2·r_c.

### 2.4 Критическое пересыщение (мёртвая зона)

Граница мёртвой зоны σ_d — пересыщение, при котором 2·r_c = L:

```
σ_d = 2·γ·Ω / (k_B·T·L)
```

Среднее расстояние между примесями через поверхностную плотность n_s:

```
L = 1/√n_s
```

Подставляя:

```
σ_d = 2·γ·Ω·√n_s / (k_B·T)
```

**Это центральный результат CV-модели**: ширина мёртвой зоны ∝ √n_s.

### 2.5 Изотерма Ленгмюра для поверхностного покрытия

Связь поверхностной плотности n_s с объёмной концентрацией примеси c_imp:

```
θ = n_s/n_max = K·c_imp / (1 + K·c_imp)
```

где K — константа равновесия адсорбции, n_max — максимальная плотность мест.

При малых концентрациях (K·c_imp ≪ 1):

```
n_s ≈ n_max · K · c_imp
```

Следовательно: **σ_d ∝ √c_imp** при малых концентрациях примеси.

### 2.6 Температурная зависимость

Мёртвая зона зависит от T через несколько факторов:

1. r_c = γ·Ω/(k_B·T·σ) — прямая зависимость 1/T
2. γ(T) — свободная энергия ступени обычно уменьшается с T
3. K(T) = K₀·exp(-ΔH_ads/(k_B·T)) — константа Ленгмюра, Аррениусовская зависимость

Суммарный эффект: мёртвая зона **сужается** при повышении температуры из-за:
- Фактора 1/T
- Усиленной десорбции примесей (K(T) уменьшается)

### 2.7 Скорость роста с CV-моделью

Объединяя CV скорость ступени с механизмом спирального роста (BCF):

```
R = R₀ · (1 - σ_d/σ)²    при σ > σ_d
R = 0                      при σ ≤ σ_d
```

Квадратичная зависимость возникает потому, что и скорость ступени, и плотность ступеней подвержены примесному пиннингу.

### 2.8 Экспериментальная валидация на KDP

**Land, De Yoreo et al. (Nature, 1999)**: AFM наблюдение динамики ступеней на KDP{100} в присутствии Fe³⁺:
- Ширина мёртвой зоны ∝ √[Fe³⁺] — подтверждение CV-предсказания
- Выход из мёртвой зоны через формирование макроступеней (пучки ~10-12 элементарных ступеней)
- CV-модель хорошо работает вблизи границы мёртвой зоны, но завышает ингибирование при больших пересыщениях

**Kubota (1997)**: Показал, что для некоторых комбинаций примесь-грань мёртвая зона отсутствует, что противоречит простой CV-модели → модифицированные версии с коэффициентом эффективности примеси.

## 3. Теория спирального роста BCF — полный вывод

### 3.1 Шаг 1: Поверхностная диффузия

Адатом на террасе диффундирует с коэффициентом D_s. Стационарная концентрация n(x) на террасе шириной λ₀ между параллельными ступенями:

```
D_s · d²n/dx² - n/τ_s + J = 0
```

где τ_s — среднее время жизни на поверхности до десорбции, J — поток из раствора.

Характерная длина поверхностной диффузии:

```
x_s = √(D_s · τ_s)
```

### 3.2 Шаг 2: Скорость ступени

Скорость ступени пропорциональна чистому потоку адатомов с обеих сторон:

```
v_step = Ω · D_s · [dn/dx|_right - dn/dx|_left]
```

Для ступеней с межступенным расстоянием λ₀, решая уравнение диффузии с граничными условиями:

```
v_step = 2·Ω·β·(n_eq·σ)·x_s·tanh(λ₀/(2·x_s))
```

где β — кинетический коэффициент присоединения к ступени, n_eq — равновесная плотность адатомов.

### 3.3 Шаг 3: Геометрия спирали

Винтовая дислокация создаёт спираль с межступенным расстоянием:

```
λ₀ = 4π·r_c / C₁ = 4π·γ·Ω / (C₁·k_B·T·σ)
```

где C₁ ≈ 19 — геометрическая константа для Архимедовой спирали.

При малом σ → λ₀ велико; при большом σ → λ₀ мало.

### 3.4 Шаг 4: Нормальная скорость роста

```
R = v_step · h / λ₀
```

где h — высота ступени.

### 3.5 Шаг 5: Формула BCF

Определяя σ* = π·x_s·C₁·k_B·T / (2·γ·Ω):

```
R = A · σ² · tanh(σ*/σ)
```

где A — составная константа, включающая D_s, n_eq, Ω, h, γ, x_s, T.

### 3.6 Предельные случаи

**Малое пересыщение** (σ ≪ σ*): tanh(σ*/σ) → 1, поэтому **R ~ A·σ²** (параболический закон)

**Большое пересыщение** (σ ≫ σ*): tanh(σ*/σ) → σ*/σ, поэтому **R ~ A·σ*·σ** (линейный закон)

Пересыщение σ* разделяет два режима.

### 3.7 Модель 2D нуклеации

Для поверхностей без винтовых дислокаций рост требует 2D островковой нуклеации:

```
R = C · exp(-π·γ²·Ω / (h·(k_B·T)²·σ)) = C · exp(-B/σ)
```

B = π·γ²·Ω / (h·(k_B·T)²) — нуклеационный барьер.

### 3.8 Модель Birth-and-Spread (B+S)

При множественной 2D нуклеации на растущей поверхности:

```
R = C' · σ^(5/6) · exp(-B/(3·σ))
```

Степень 5/6 от σ возникает из связи между скоростью нуклеации и латеральным распространением.

### 3.9 Почему используется эмпирический степенной закон?

R = s₀·(σ - s₁)^w используется потому, что:

1. Включает мёртвую зону (s₁ > 0 — порог CV)
2. На ограниченном экспериментальном диапазоне σ и BCF, и B+S аппроксимируются степенным законом
3. Показатель w — диагностика: w ≈ 2 → BCF спиральный рост; w > 2 → 2D нуклеация; w ≈ 1 → грубый рост или высоко-σ BCF
4. Алгебраически проще для фитинга (3 параметра vs физически осмысленные но трудно определяемые параметры BCF)

Степенной закон — **эффективная модель**, поглощающая сложность физики в эмпирические параметры.

### 3.10 Механизм роста KDP

- **{101} призматические грани**: Преимущественно спиральный (BCF) рост, w ≈ 2
- **{100} грани**: w от 0.87 до 4.12 в разных экспериментах, часто w > 2 → полинуклеарный рост
- При малом σ на {101}: параболический (BCF-like)
- При большом σ: переход к линейному или 2D нуклеации

# Часть II: Статистические методы — полные выкладки

## 4. NLME (Nonlinear Mixed Effects Models)

### 4.1 Спецификация модели

Для эксперимента i, наблюдения j:

```
y_ij = f(x_ij, φ_i) + ε_ij
```

где:
- φ_i = β + b_i (индивидуальные параметры = фиксированные + случайные эффекты)
- b_i ~ N(0, Ω) (случайные эффекты, p-мерные)
- ε_ij ~ N(0, σ²) (ошибка наблюдения)

Для нашей задачи: f = s₀_i · max(σ_ij - s₁_i, 0)^w, где s₀_i = μ_s₀ + η_s₀_i, s₁_i = μ_s₁ + η_s₁_i.

### 4.2 Маргинальное правдоподобие

Поскольку b_i ненаблюдаемы, маргинальное правдоподобие интегрирует их:

```
L(θ) = ∏ᵢ₌₁ᴺ ∫ p₁(dᵢ|θ, ηᵢ) · p₂(ηᵢ|θ) dηᵢ
```

где:
- p₁ = ∏ⱼ (2πσ²)^(-1/2) · exp(-ε²_ij/(2σ²))
- p₂ = (2π)^(-p/2) · |Ω|^(-1/2) · exp(-η'ᵢΩ⁻¹ηᵢ/2)

Этот интеграл **не имеет аналитического решения** для нелинейного f → нужны приближения.

### 4.3 Аппроксимация Лапласа

Подынтегральное выражение: exp(lᵢ(ηᵢ)), где:

```
lᵢ = -1/2 · Σⱼ[ε²_ij/σ² + ln(2πσ²)] - 1/2 · η'ᵢΩ⁻¹ηᵢ - 1/2 · ln|2πΩ|
```

Находим моду η*ᵢ = argmax lᵢ(ηᵢ). Разложение Тейлора 2-го порядка:

```
lᵢ(ηᵢ) ≈ lᵢ(η*ᵢ) + 1/2 · (ηᵢ - η*ᵢ)' Hᵢ(η*ᵢ) (ηᵢ - η*ᵢ)
```

где Hᵢ — гессиан lᵢ. Гауссов интеграл:

```
∫ exp(lᵢ) dηᵢ ≈ exp(lᵢ(η*ᵢ)) · (2π)^(p/2) / √det(-Hᵢ(η*ᵢ))
```

### 4.4 FOCE (First-Order Conditional Estimation)

Линеаризация f вокруг условной моды η*ᵢ:

```
f(x_ij, β + bᵢ) ≈ f(x_ij, β + η*ᵢ) + (∂f/∂η)|_{η=η*ᵢ} · (ηᵢ - η*ᵢ)
```

Преобразует нелинейную модель в локально линейную смешанную → интеграл вычисляется аналитически на каждой итерации.

### 4.5 EM-алгоритм для NLME

**E-шаг**: Вычислить E[bᵢ|yᵢ; θ⁽ᵏ⁾] и E[bᵢb'ᵢ|yᵢ; θ⁽ᵏ⁾]:
- Условная ковариация: Γᵢ = (A'ᵢAᵢ/σ² + Ω⁻¹)⁻¹
- Условное среднее: μᵢ = Γᵢ A'ᵢ(yᵢ - Xᵢβ)/σ²
- E[ηᵢη'ᵢ|yᵢ] = Γᵢ + μᵢμ'ᵢ

**M-шаг**: Обновить параметры:
- β⁽ᵏ⁺¹⁾ = (X'X)⁻¹X'(y - A·E[η|y])
- Ω⁽ᵏ⁺¹⁾ = (1/N)·Σᵢ E[ηᵢη'ᵢ|y; θ⁽ᵏ⁾]
- σ²⁽ᵏ⁺¹⁾ = (1/n)·[||y - Xβ||² + Σᵢ tr(A'ᵢAᵢ·E[ηᵢη'ᵢ|y]) - 2Σᵢ(yᵢ-Xᵢβ)'Aᵢ·E[ηᵢ|y]]

### 4.6 NLME vs Pooling vs Averaging

- **Pooling** (игнорировать группировку): Все данные как от одного субъекта. Смещённо если Ω ≠ 0.
- **Averaging** (фит каждого, потом среднее): Теряет точность, не учитывает корреляции, не работает при малом числе точек на эксперимент.
- **NLME**: Оптимально при реальной между-экспериментной вариативности. Заимствует силу через shrinkage. Работает с неполными данными.

## 5. Иерархическая Байесовская модель

### 5.1 Полная спецификация

**Уровень данных**: Y_ij | θᵢ ~ p(Y_ij | θᵢ)

**Уровень экспериментов**: θᵢ | μ, τ ~ N(μ, τ²)

**Гиперприоры**:
- μ ~ N(μ₀, σ₀²) (например, μ₀ = 0.3, σ₀ = 0.5 для s₀)
- τ ~ Half-Cauchy(0, s)
- σ ~ Half-Cauchy(0, s₁)

### 5.2 Постериор через теорему Байеса

```
p(μ, τ, θ₁,...,θₙ | data) ∝ [∏ᵢ ∏ⱼ p(y_ij|θᵢ)] · [∏ᵢ p(θᵢ|μ,τ)] · p(μ) · p(τ)
```

### 5.3 Shrinkage (усадка)

Постериорное среднее θᵢ — взвешенное среднее:

```
E(θᵢ|y) = (1 - λᵢ)·ȳᵢ + λᵢ·μ
```

где λᵢ = τ² / (σ²/nᵢ + τ²) — доля shrinkage.

**Физический смысл**: Эксперименты с малым числом данных (nᵢ мало) сильнее «притягиваются» к популяционному среднему μ.

### 5.4 Credible vs Confidence Intervals

- **Credible interval** (Баесовский): P(θ ∈ [a,b] | data) = 0.95. Прямое вероятностное утверждение о θ.
- **Confidence interval** (частотный): При повторной выборке 95% интервалов содержат истинное θ. НЕ утверждение о θ для данных.

### 5.5 Posterior Predictive для нового эксперимента

```
p(y_new|data) = ∫∫ p(y_new|θ_new) · p(θ_new|μ,τ) · p(μ,τ|data) dθ_new dμdτ
```

Практически через MCMC: для каждого сэмпла (μ⁽ˢ⁾, τ⁽ˢ⁾) → θ_new⁽ˢ⁾ ~ N(μ⁽ˢ⁾, τ⁽ˢ⁾) → y_new⁽ˢ⁾ ~ p(y|θ_new⁽ˢ⁾).

**Источники**:
- Bayesian Hierarchical Modeling: https://bayesball.github.io/BOOK/bayesian-hierarchical-modeling.html
- Hierarchical Bayesian in Experimental Psychopathology: https://pmc.ncbi.nlm.nih.gov/articles/PMC8634778/

## 6. Bootstrap для нелинейной регрессии

### 6.1 Residual Bootstrap

1. Фит модели → θ̂ и остатки eᵢ = yᵢ - f(xᵢ, θ̂)
2. Центрирование: eᵢᶜ = eᵢ - mean(e)
3. Для b = 1,...,B:
   a. Ресэмпл: e*₁,...,e*ₙ с возвращением из {eᵢᶜ}
   b. Bootstrap данные: y*ᵢ = f(xᵢ, θ̂) + e*ᵢ
   c. Перефит → θ̂*⁽ᵇ⁾
4. 95% CI = [quantile₀.₀₂₅(θ̂*), quantile₀.₉₇₅(θ̂*)]

### 6.2 Почему residual, а не pairs bootstrap?

**Residual**: сохраняет структуру design matrix (xᵢ фиксированы). Предполагает гомоскедастичность.

**Pairs**: ресэмплирует (xᵢ, yᵢ) вместе. Может дать xᵢ вне наблюдаемого диапазона → проблемы фита нелинейных моделей.

Для регрессии с контролируемыми предикторами (наш случай — dT контролируется) предпочтителен residual bootstrap.

### 6.3 BCa (Bias-Corrected and Accelerated)

Формула конечных точек:

```
θ̂_BCa(α) = G⁻¹(Φ(z₀ + (z₀ + z_α)/(1 - â·(z₀ + z_α))))
```

**Коррекция смещения z₀**:
```
z₀ = Φ⁻¹(G(θ̂))
```

**Фактор ускорения â** (через jackknife):
```
â = (1/6) · Σ(dᵢ³) / (Σ(dᵢ²))^(3/2)
```

где dᵢ = θ̂₍ᵢ₎ - θ̂₍.₎ — jackknife influence values.

BCa даёт **второго порядка точность**: ошибка покрытия O(1/n) vs O(1/√n) для стандартных интервалов.

### 6.4 Сравнение с Fisher Information Matrix CI

- **FIM CI**: Cov(θ̂) ≈ (J'J)⁻¹·σ²_ε. Предполагает локальную линейность, гауссовы остатки. Быстро. Может быть неточным (реальное покрытие 95% CI может быть лишь 75%).
- **Bootstrap CI**: Без предположений о распределении. Обрабатывает нелинейность. BCa — инвариантен к трансформации, возможно асимметричен.

**Источники**:
- BCa construction: https://pmc.ncbi.nlm.nih.gov/articles/PMC7958418/
- Bootstrap for nonlinear models: https://cran.r-project.org/web/packages/nlraa/vignettes/Bootstrapping.html

## 7. Мета-анализ кинетических кривых

### 7.1 Модель фиксированных эффектов

```
yᵢ = θ + εᵢ,  εᵢ ~ N(0, σ²ᵢ)
```

Оптимальная оценка — обратно-дисперсионное взвешенное среднее:

```
θ̂_FE = Σ(wᵢ·yᵢ) / Σ(wᵢ),  где wᵢ = 1/σ²ᵢ
Var(θ̂_FE) = 1/Σ(wᵢ)
```

### 7.2 Модель случайных эффектов

```
yᵢ = θᵢ + εᵢ,  θᵢ ~ N(μ, τ²),  εᵢ ~ N(0, σ²ᵢ)
```

Маргинально: yᵢ ~ N(μ, σ²ᵢ + τ²)

Веса: w*ᵢ = 1/(σ²ᵢ + τ²)

```
μ̂_RE = Σ(w*ᵢ·yᵢ) / Σ(w*ᵢ)
```

### 7.3 Q-статистика Кокрейна

```
Q = Σ wᵢ·(yᵢ - θ̂_FE)²
```

При H₀ (однородность): Q ~ χ²(k-1)

### 7.4 Оценка DerSimonian-Laird для τ²

```
τ²_DL = max(0, (Q - (k-1)) / c)
```

где c = Σ(wᵢ) - Σ(w²ᵢ)/Σ(wᵢ)

### 7.5 I² — мера гетерогенности

```
I² = max(0, 100% · (Q - df) / Q)
```

I² = 0%: нет гетерогенности; 25%: низкая; 50%: умеренная; 75%: высокая.

**Источники**:
- Meta-Analysis in Clinical Trials: https://pmc.ncbi.nlm.nih.gov/articles/PMC4639420/
- DerSimonian-Laird 2-step: https://sammancuso.com/2020/02/12/meta-analysis-with-the-2-step-dersimonian-and-laird-estimator/

## 8. Fisher Information Matrix для степенного закона

### 8.1 Общая FIM для нелинейной регрессии

Для модели yᵢ = f(xᵢ, θ) + εᵢ, εᵢ ~ N(0, σ²):

```
I(θ) = J'J / σ²
```

где J — якобиан: J_ij = ∂f(xᵢ,θ)/∂θⱼ

```
Cov(θ̂) ≈ (J'J)⁻¹ · σ²
```

### 8.2 Граница Крамера-Рао (доказательство)

Пусть T(X) — несмещённая оценка, s(θ) = ∂ln f(X;θ)/∂θ — score.

**Факт**: E[s(θ)] = 0 (математическое ожидание score = 0).

**Доказательство**: ∫[∂f/∂θ]/f · f dx = ∂/∂θ ∫f dx = ∂/∂θ · 1 = 0.

**Применяем Коши-Шварц**: |Cov[T(X), s(θ)]|² ≤ Var[T(X)] · Var[s(θ)]

Числитель: Cov[T, s] = ∂E[T]/∂θ = 1 (для несмещённой оценки).

Знаменатель: Var[s] = I(θ).

**Результат**: **Var[T(X)] ≥ 1/I(θ)**

### 8.3 Для R = s₀·(σ - s₁)^w

Компоненты якобиана:

```
∂R/∂s₀ = (σ - s₁)^w
∂R/∂s₁ = -s₀·w·(σ - s₁)^(w-1)
∂R/∂w  = s₀·(σ - s₁)^w · ln(σ - s₁)
```

**Почему s₀ и s₁ коррелированы**: ∂R/∂s₀ и ∂R/∂s₁ имеют похожую функциональную форму (оба ∝ (σ-s₁)^something) → почти коллинеарность в J → большие внедиагональные элементы (J'J)⁻¹.

### 8.4 Profile Likelihood

Для отдельного параметра θₖ:

```
L_P(θₖ) = max_{θ₋ₖ} L(θₖ, θ₋ₖ)
```

CI: {θₖ : 2·[l(θ̂) - l_P(θₖ)] ≤ χ²₁,₁₋α}

Profile likelihood CI: инвариантны к трансформации, потенциально асимметричны, точнее для малых выборок.

**Источники**:
- Fisher Score and Information: https://jaketae.github.io/study/fisher/
- Proof of Cramer-Rao: https://gregorygundersen.com/blog/2019/11/27/proof-crlb/
- Fisher Information (Stanford): https://web.stanford.edu/class/archive/stats/stats200/stats200.1172/Lecture15.pdf
- Profile Likelihood: https://journals.sagepub.com/doi/pdf/10.1177/1536867X0700700305

## 9. Робастная консенсусная оценка tₙ

### 9.1 Взвешенная медиана

Breakdown point = 50%: устойчива к тому, что до 50% данных могут быть произвольно испорчены.

### 9.2 MAD (Median Absolute Deviation)

```
MAD = median(|Xᵢ - median(X)|)
σ̂ = 1.4826 · MAD
```

Константа 1.4826 = 1/Φ⁻¹(3/4) обеспечивает E[1.4826·MAD] = σ для X ~ N(μ,σ²).

### 9.3 Критерий выбросов

Флаг xᵢ как выброс если |xᵢ - median(X)| / MAD > k (k = 2.5 или 3).

### 9.4 Тест Граббса

Статистика: G = max|Yᵢ - Ȳ| / s

Критическое значение: отвергаем H₀ если G > ((N-1)/√N)·√(t²_{α/(2N),N-2} / (N-2+t²_{α/(2N),N-2}))

### 9.5 Тест однородности tₙ (Q Кокрейна)

Если Q > χ²(k-1, 0.95), то tₙ не одинаков → раствор деградировал между экспериментами.

### 9.6 Модель дрейфа tₙ

Если tₙ дрейфует (старение раствора):

```
tₙ(t) = tₙ₀ + β·t
```

Оценка β → предсказание tₙ для будущих экспериментов.

# Часть III: ML/AI методы — математические основы

## 10. Simulation-Based Inference (SBI)

### 10.1 Проблема

Дан симулятор θ → x через латентные z:

```
p(x|θ) = ∫ p(x,z|θ) dz
```

Этот интеграл как правило неаналитичен. Цель:

```
p(θ|x) = p(x|θ)·p(θ) / p(x)
```

### 10.2 Neural Posterior Estimation (NPE)

Обучаем условный density estimator q_φ(θ|x) ≈ p(θ|x). Цель обучения:

```
L_NPE = -E_{p(θ,x)} [log q_φ(θ|x)]
```

что эквивалентно минимизации KL(p(θ|x) || q_φ(θ|x)) усреднённой по p(x).

### 10.3 Normalizing Flows как density estimators

Обратимое отображение f: z → x преобразует базовое распределение p_z(z) (гауссиан):

```
p_x(x) = p_z(f⁻¹(x)) · |det(∂f⁻¹/∂x)|
```

Для композиции K преобразований f = f_K ∘ ... ∘ f₁:

```
log p(x) = log p_z(z₀) - Σₖ₌₁ᴷ log|det(∂fₖ/∂z_{k-1})|
```

### 10.4 Neural Spline Flows (Rational-Quadratic)

В k-м бине, с нормализованной координатой ξ = (x-xₖ)/(x_{k+1}-xₖ):

```
fₖ(x) = yₖ + (y_{k+1}-yₖ)·[sₖξ² + δₖξ(1-ξ)] / [sₖ + (δ_{k+1}+δₖ-2sₖ)ξ(1-ξ)]
```

где sₖ = (y_{k+1}-yₖ)/(x_{k+1}-xₖ), δₖ = производная в узле k.

Производная (log-det-Jacobian):

```
dfₖ/dx = sₖ²·(δ_{k+1}ξ² + 2sₖξ(1-ξ) + δₖ(1-ξ)²) / [sₖ + (δ_{k+1}+δₖ-2sₖ)ξ(1-ξ)]²
```

Обратная: решение квадратного уравнения (аналитически).

**Свойства**: Монотонна по построению, аналитически обратима, дифференцируема.

### 10.5 Sequential NPE (SNPE-C/APT)

В раунде r, симуляция из proposal π_r(θ):

```
L_SNPE-C = -E_{π_r(θ)} [log(q_φ(θ|x) · p(θ) / π_r(θ))]
```

### 10.6 Почему это трансформативно для нашей задачи

1. **Раскрывает корреляции**: p(tₙ, s₁ | signal) показывает, что tₙ и s₁ коррелированы — grid search это скрывает
2. **Мгновенный inference**: Одно обучение → каждый новый сигнал за <1 сек
3. **Полная неопределённость**: Не точечная оценка, а распределение
4. **Заменяет весь pipeline**: Сигнал → постериор параметров напрямую

**Источники**:
- Cranmer et al. (2020) PNAS: https://www.pnas.org/doi/10.1073/pnas.1912789117
- Neural Spline Flows (Durkan et al., 2019): https://arxiv.org/abs/1906.04032
- Normalizing Flows review JMLR: https://jmlr.org/papers/volume22/19-1028/19-1028.pdf
- SBI tutorial: https://astroautomata.com/blog/simulation-based-inference/

## 11. Kolmogorov-Arnold Networks (KAN)

### 11.1 Теорема Колмогорова-Арнольда

**Теорема (Колмогоров, 1957; Арнольд, 1958)**. Для любой непрерывной f: [0,1]ⁿ → ℝ существуют непрерывные одномерные φⱼ и ψ_{i,j} такие что:

```
f(x₁,...,xₙ) = Σⱼ₌₀²ⁿ φⱼ(Σᵢ₌₁ⁿ ψ_{i,j}(xᵢ))
```

### 11.2 Архитектура KAN с B-сплайнами

Слой KAN K_{m,n}: ℝⁿ → ℝᵐ:

```
(K_{m,n}(x))ᵢ = Σⱼ₌₁ⁿ Φ_{i,j}(xⱼ)
```

Каждая обучаемая активация:

```
Φ_{i,j}(x) = w^(b)_{i,j} · SiLU(x) + w^(s)_{i,j} · s_{i,j}(x)
```

B-сплайн компонента:

```
s_{i,j}(x) = Σₕ₌₁ᴳ c^h_{i,j} · B^h_{i,j}(x)
```

G = число интервалов сетки, B^h — базисные функции B-сплайна порядка k (обычно k=3, кубические).

### 11.3 Функция потерь с регуляризацией

```
L = L_train + λ₁·||Φ||₁ + λ₂·S(Φ)
```

L1 на активациях поощряет разреженность, энтропийная регуляризация S поощряет использование немногих связей.

### 11.4 Извлечение символических формул

После обучения каждую выученную активацию можно заменить символьной функцией из библиотеки {sin, cos, exp, x², √, ...} по R² к выученному сплайну.

### 11.5 Почему KAN избегает проклятия размерности

Для класса композиционных функций (представимых KA-теоремой) глубокие KAN размера O(ε⁻¹/²) достигают ошибки ε, vs O(ε⁻ⁿ) для мелких сетей. KAN нужно аппроксимировать только 1D функции → число параметров растёт **линейно** с размерностью n.

### 11.6 KAN-ODE для динамических систем

```
dx/dt = KAN(x, t)
```

Обучает правую часть ODE как KAN → символьная извлекаемая формула кинетики.

Применение к уравнению Аллена-Кана (фазовое разделение) — близко к росту кристаллов.

**Источники**:
- KAN (ICLR 2025): https://arxiv.org/abs/2404.19756
- The Annotated KAN: https://alexzhang13.github.io/blog/2024/annotated-kan/
- KAN-ODE: https://arxiv.org/abs/2407.04192
- KA theorem revisited: https://arxiv.org/abs/2007.15884

## 12. Physics-Informed Neural Networks (PINNs)

### 12.1 Формулировка

Для PDE системы на области Ω:

```
N[u](x,t) = f(x,t)    — уравнение
B[u](x,t) = g(x,t)    — граничные условия
u(x,0) = h(x)          — начальные условия
```

Аппроксимируем u нейросетью u_θ. Функция потерь PINN:

```
L(θ) = λ_data·L_data + λ_phys·L_phys + λ_BC·L_BC + λ_IC·L_IC
```

```
L_data  = (1/N_d)·Σ|u_θ(xᵢ,tᵢ) - u_obs_i|²
L_phys  = (1/N_r)·Σ|N[u_θ](x^r_i,t^r_i) - f|²
L_BC    = (1/N_b)·Σ|B[u_θ] - g|²
L_IC    = (1/N_0)·Σ|u_θ(x_i,0) - h|²
```

Градиенты N[u_θ] вычисляются через автоматическое дифференцирование.

### 12.2 NTK-анализ (Wang et al., 2022)

NTK матрица: K(θ) = J(θ)·J(θ)' где J — якобиан вектора невязок по θ.

Собственные значения K определяют скорости сходимости разных компонент потерь. Если собственные значения для L_phys намного меньше чем для L_data → физика сходится медленно (**spectral bias**).

**Адаптивные веса**: выбрать λᵢ обратно пропорционально соответствующим собственным значениям NTK.

### 12.3 Self-Adaptive PINN (SA-PINN)

Поточечные обучаемые веса на каждой коллокационной точке:

```
L_r(w,λ_r) = (1/2)·Σᵢ m(λ_{r,i})·|N[u_θ] - f|²
```

Двойная оптимизация:
- Веса сети: w_{k+1} = wₖ - η·∇_w L (gradient descent)
- Адаптивные веса: λ_{k+1} = λₖ + ρ·∇_λ L (gradient **ascent**)

Точки с большими невязками автоматически получают бóльшие веса.

### 12.4 Режимы отказа

1. **Жёсткие уравнения**: Быстро-медленное разделение шкал
2. **Резкие градиенты**: На границе мёртвой зоны R резко переходит 0 → R > 0
3. **Спектральное смещение**: Сеть сначала учит низкочастотные компоненты

**Источники**:
- When and why PINNs fail: https://arxiv.org/abs/2007.14527
- Self-Adaptive PINNs: https://arxiv.org/abs/2009.04544
- PINN failure modes (NeurIPS 2021)
- PIRNN for crystallization: https://www.sciencedirect.com/science/article/pii/S0263876223001442

## 13. Conformal Prediction — гарантия покрытия

### 13.1 Split Conformal Prediction

1. Фит модели μ̂ на training subset
2. Остатки на калибровке: Rᵢ = |yᵢ - μ̂(xᵢ)|
3. Квантиль: q = Quantile_{1-α} из {R₁,...,Rₙ, +∞}
4. Предсказательный интервал: **C(x_{n+1}) = [μ̂(x_{n+1}) - q, μ̂(x_{n+1}) + q]**

### 13.2 Доказательство гарантии покрытия

**Теорема**: Если (X₁,Y₁),...,(Xₙ,Yₙ),(X_{n+1},Y_{n+1}) обменимы, то:

```
P(Y_{n+1} ∈ C(X_{n+1})) ≥ 1 - α
```

**Доказательство**: По обменимости, R_{n+1} с равной вероятностью занимает любую ранговую позицию среди R₁,...,Rₙ,R_{n+1}. Вероятность того что R_{n+1} превысит ⌈(1-α)(n+1)⌉-ю наименьшую не более α. □

### 13.3 Adaptive Conformal Inference (Gibbs & Candès, NeurIPS 2021)

Для нестационарных данных α_t обновляется онлайн:

```
α_{t+1} = αₜ + γ·(α - errₜ)
```

где errₜ = 𝟙{Yₜ ∉ Cₜ(Xₜ)}.

**Теорема**: Долгосрочное покрытие **независимо от процесса генерации данных**:

```
|1/T · Σerrₜ - α| ≤ 1/(γT) + γ/2
```

Оптимизируя γ = √(2/T) → скорость O(1/√T).

### 13.4 CPTC для change-points

Интегрирует state-switching модель с online CP:

- Отдельные предсказательные интервалы для каждого режима (рост / мёртвая зона)
- Адаптивное обновление α для каждого режима
- Предсказательные интервалы расширяются на переходе → принципиальная неопределённость tₙ

**Источники**:
- Gibbs & Candès (2021) NeurIPS: https://papers.neurips.cc/paper_files/paper/2021/file/0d441de75945e5acbc865406fc9a2559-Paper.pdf
- CPTC (2025): https://arxiv.org/abs/2509.02844
- Conformal Beyond Exchangeability: https://arxiv.org/html/2202.13415v5

## 14. Symbolic Regression (PySR / SINDy)

### 14.1 Генетическое программирование

Математические выражения — деревья. Внутренние узлы = операторы (+,-,*,/,sin,exp,log,^), листья = переменные или константы.

**Цикл эволюции**:
1. Инициализация популяции N случайных деревьев
2. Фитнес: f(expr) = MSE(expr, data) + λ·complexity(expr)
3. Селекция: турнирная
4. Кроссовер: обмен поддеревьями
5. Мутации: замена оператора, вставка, удаление
6. Повтор G поколений

### 14.2 Парето-фронт

Две цели: минимизировать ошибку и сложность. Парето-фронт = множество недоминируемых решений.

Выбор модели: AIC, BIC, или PySR score = Δ(log_loss)/Δ(complexity).

### 14.3 SINDy: Sparse Identification of Nonlinear Dynamics

```
Ẋ = Θ(X) · Ξ
```

Θ — библиотека функций-кандидатов, Ξ — разреженная матрица коэффициентов.

```
min_Ξ ||Ẋ - Θ·Ξ||₂² + λ·||Ξ||₁
```

L1 штраф → разреженность → выбор только релевантных членов.

### 14.4 Применение к кинетике KDP

Библиотека SINDy для нашей задачи:
- σⁿ (n = 1, 2, 3)
- (σ - σ_d)ⁿ
- exp(-B/σ)
- σ·tanh(C/σ) (BCF-like)
- T-зависимые предэкспоненты

**Nyande et al. (2024, AIChE J.)**: SINDy успешно восстановил уравнения кинетики кристаллизации из зашумлённых данных. «Good agreement between data-driven model and experimental data.»

## 15. Neural ODE — метод сопряжённых

### 15.1 Прямая задача

```
dz/dt = f_θ(z(t), t),  z(t₀) = z₀
```

### 15.2 Сопряжённое состояние

```
λ(t) = dL/dz(t)
```

### 15.3 Вывод сопряжённого ODE

Лагранжиан с ODE-ограничением:

```
J = L(z(t₁)) + ∫ₜ₀^t₁ λ(t)'·(f_θ(z,t) - dz/dt) dt
```

Интегрирование по частям λ'·dz/dt:

```
∫ λ'·dz/dt dt = [λ'z]_{t₀}^{t₁} - ∫ (dλ/dt)'·z dt
```

Приравнивая вариацию δJ/δz(t) = 0:

```
dλ/dt = -λ(t)'·(∂f/∂z)
```

Это **сопряжённое ODE**, решается **назад** от t₁ к t₀.

### 15.4 Градиент по параметрам

```
dL/dθ = -∫_{t₁}^{t₀} λ(t)'·(∂f/∂θ) dt
```

### 15.5 Эффективность по памяти

- **Backprop через решатель**: O(T) памяти
- **Метод сопряжённых**: O(1) памяти — хранит только z(t₁) и пересчитывает z(t) на обратном проходе

### 15.6 Применение к интерферометрическому сигналу

Сигнал: I(t) = A(t)·sin(φ(t)) + noise

Neural ODE моделирует:

```
dφ/dt = (4π·n/λ_light) · R(σ(t), θ)
```

**Преимущество**: Избегает явного извлечения фазы (Hilbert, arcsin) — работает напрямую с сырым сигналом.

**Источники**:
- Chen et al. (NeurIPS 2018): https://arxiv.org/abs/1806.07366
- Adjoint via Lagrange multipliers: https://vaipatel.com/posts/deriving-the-adjoint-equation-for-neural-odes-using-lagrange-multipliers/
- Neural ODE for parameter estimation (2024): https://www.sciencedirect.com/science/article/abs/pii/S1568494624014236

## 16. Gaussian Process Regression для кинетических кривых

### 16.1 Определение GP

```
f(x) ~ GP(m(x), k(x,x'))
```

Для конечного набора: f ~ N(m(X), K(X,X))

### 16.2 Постериор (полная матричная алгебра)

Наблюдения: y = f(X) + ε, ε ~ N(0, σ²ₙI)

Совместное распределение train + test:

```
[y  ]     ([m(X)  ], [K(X,X)+σ²ₙI    K(X,X*) ])
[f* ] ~ N ([m(X*)], [K(X*,X)          K(X*,X*)])
```

Условное (через формулу для многомерного гауссиана):

```
μ* = m(X*) + K(X*,X)·[K(X,X)+σ²ₙI]⁻¹·(y - m(X))
Σ* = K(X*,X*) - K(X*,X)·[K(X,X)+σ²ₙI]⁻¹·K(X,X*)
```

### 16.3 Маргинальное правдоподобие

```
log p(y|X,θ) = -1/2·y'K_y⁻¹y - 1/2·log|K_y| - n/2·log(2π)
```

Три члена: подгонка к данным, штраф за сложность (бритва Оккама), нормировка.

### 16.4 Ядра Matérn

```
k_ν(r) = σ²_f · 2^(1-ν)/Γ(ν) · (√(2ν)·r/l)^ν · K_ν(√(2ν)·r/l)
```

| ν | Формула | Гладкость |
|---|---------|-----------|
| 1/2 | σ²_f·exp(-r/l) | Непрерывная, не дифференцируемая |
| 3/2 | σ²_f·(1+√3·r/l)·exp(-√3·r/l) | Однократно дифференцируемая |
| 5/2 | σ²_f·(1+√5·r/l+5r²/(3l²))·exp(-√5·r/l) | Дважды дифференцируемая |
| ∞ | σ²_f·exp(-r²/(2l²)) | Бесконечно гладкая (RBF) |

**Для кинетики**: ν=5/2 предпочтительно — кинетические кривые гладкие, но не бесконечно (мёртвая зона создаёт излом).

### 16.5 Физически информированная средняя функция

```
m(σ) = s₀·max(σ - s₁, 0)^w
```

GP обучает **отклонения** от физической модели → физика описывает глобальный тренд, GP — локальные и систематические отклонения.

### 16.6 Multi-Output GP для объединения экспериментов

Linear Model of Coregionalization (LMC):

```
f_j(x) = Σq a_{j,q}·u_q(x),  u_q ~ GP(0, k_q)
```

Кросс-ковариация:

```
Cov(f_i(x), f_j(x')) = Σq a_{i,q}·a_{j,q}·k_q(x,x')
```

Позволяет **заимствовать силу** между экспериментами при моделировании per-experiment вариативности.

**Источники**:
- GP Regression tutorial: https://arxiv.org/abs/2502.03090
- Physics-informed GP: https://link.aps.org/pdf/10.1103/PhysRevAccelBeams.24.072802
- Multi-output GP (PyMC): https://www.pymc.io/projects/examples/en/latest/gaussian_processes/MOGP-Coregion-Hadamard.html
- Kernel Cookbook: https://www.cs.toronto.edu/~duvenaud/cookbook/

## 18. PELT Change-Point Detection — алгоритм

### 18.1 Задача

Найти changepoints τ₁,...,τₘ минимизирующие:

```
Σₖ₌₀ᵐ C(y_{τₖ+1:τ_{k+1}}) + m·β
```

### 18.2 Динамическое программирование

```
Qₜ = min_{0≤s<t} [Qₛ + C(y_{s+1:t}) + β],  Q₀ = -β
```

Наивно O(n²).

### 18.3 PELT pruning

Если Qτ + C(y_{τ+1:t}) + κ ≥ Qₜ, то τ НИКОГДА не будет оптимальным для любого T > t.

**Доказательство**: Добавляем C(y_{t+1:T}) + β к обеим сторонам. Фундаментальное неравенство C(y_{τ+1:t}) + C(y_{t+1:T}) + κ ≥ C(y_{τ+1:T}) даёт:

```
Qτ + C(y_{τ+1:T}) + β ≥ Qₜ + C(y_{t+1:T}) + β
```

т.е. τ хуже чем t для ВСЕХ будущих T. □

Для гауссовой cost: κ = 0.

**Сложность**: O(n) в среднем (при линейном росте changepoints с n).

### 18.4 Гауссова cost function

```
C(y_{s+1:t}) = Σ(yᵢ - ȳ_{s+1:t})²
```

### 18.5 Выбор штрафа

- BIC: β = 2·log(n)
- mBIC: β = 3·log(n)
- CROPS: исследует диапазон [β_min, β_max]

### 18.6 Применение к мёртвой зоне

Используем change in variance как cost. До границы мёртвой зоны — дисперсия мала (только шум). После — дисперсия растёт (осцилляции). PELT с C = n·log(σ²_MLE) обнаруживает этот переход.

**Источники**:
- PELT paper: https://arxiv.org/pdf/1101.1438
- PELT algorithms and penalties: https://www.lancaster.ac.uk/~romano/teaching/2425MATH337/4_algos_and_penalties.html
- BOCPD (Adams & MacKay): https://arxiv.org/abs/0710.3742

## 19. Soft Sensor парадигма

### 19.1 Постановка

```
y(t) = g(x₁(t), x₂(t), ..., xₚ(t)) + ε(t)
```

y — трудноизмеримый параметр (tₙ, Td, R), xᵢ — легкоизмеримые (LED1, LED2, T).

### 19.2 Feature engineering для временных рядов

- **Скользящие статистики** (окно W): μ, σ, min, max, skew, kurtosis
- **FFT features**: Доминантная частота = скорость роста
- **Wavelet коэффициенты**: CWT на выбранных масштабах
- **Lag features**: x(t), x(t-1), ..., x(t-L)

### 19.3 Bias-Variance tradeoff

```
E[(y - ĝ(x))²] = Bias²[ĝ] + Var[ĝ] + σ²_noise
```

| Модель | Bias | Variance | Для малых данных |
|--------|------|----------|-----------------|
| Linear | High | Low | OK |
| XGBoost | Low-Med | Low-Med | **Лучший выбор** |
| GP | Low | Low-Med | **Лучший + UQ** |
| NN | Low | High | Переобучение |

### 19.4 Walk-Forward Cross-Validation

```
Fold 1: Train [1,...,T₁], Test [T₁+1,...,T₁+h]
Fold 2: Train [1,...,T₁+h], Test [T₁+h+1,...,T₁+2h]
...
```
